保研夏令营的个人陈述怎么写？

匿名用户的回答

夏令营入营：清华大学深圳研究生院、上海交通大学、中科大等

最终去向：上海交通大学

作为一个绩点/排名并无优势，比赛经历乏善可陈的工科生（对工科生来说，少比赛和项目真是硬伤啊），我在申请夏令营的时候却出乎意外的顺利，我想这除了运气外，大概跟那篇看似简洁流畅、内中却自有巧思的个人自述不无关系。写个人陈述这件事，就像打桥牌，手上的牌是一定的，但打得好却可以把手上的牌面优势发挥到最大。今天我就带着大家手把手来写一篇不错的自述~

话不多说，我们就开始吧！

开写之前，我们再重温一下个人自述的要求。##各个学校都蛮大同小异，无非就是字数上的区别

请用大约1000字介绍你的学术背景、科研经历、综合素质以及对今后学习研究工作的设想和计划等。

审清题之后，我们的1000字大作也将从开头介绍，学习背景，科研经历，综合素质，未来展望，结尾6个方面开始着笔。

  1. 开头介绍
尊敬的老师：
您好！我叫xxx，现就读于xx大学xx专业，在本科多学科交叉的培养模式下，我完成了xx及xx大类的基础课程。过去两年多的时间里，我取得了优异的学习成绩并在各项活动中表现出色，获得了xx奖学金、xx优秀三好学生、xx优秀团干部等荣誉及称号，在xx科创方面也有了自己的尝试。

因为我觉得自己在学习背景，科研经历方面都不突出，都属于做的还好，但并不拔尖。优势是比较全面发展，因此在本科期间所获奖学金及称号较多##但这些奖项若是写在正文里，担心不能给老师留下深刻印象。所以，我选择了开篇亮眼，给老师打下一个“这小同学还不错”的心理基调。如果你也是这种情况，你可以从这几方面入手：

  • 堆奖项，把你最有含金量、听起来最高大上的奖项称号都拿出来，“国家奖学金”，“xx一等奖学金”“优秀学生”，或者可以侧重数量描述，如“连续三年获得...”
  • 夸学校/夸专业，“我校是第一批xx高校”，“本专业在去年的学科评估中为...”
  • 夸培养方案，“多学科交叉培养模式”，“在重数理强实验的培养模式下”

当然，如果你不是这种情况，直接清清爽爽的介绍一下自己，也很大方得体##毕竟工科老师也都挺喜欢讲效率的：

尊敬的老师：
您好！我叫我叫xxx，现就读于xx大学xx专业，已具有条件获得研究生免试攻读硕士学位的资格。下面我将从以下四方面来介绍自己。

2.学习背景

在大一、大二阶段，我学习了高数、线代、概统、大学物理、C语言、xx、xx等基础课程。大三阶段，学习了xx，xx，xx等专业课程。课程的交叉设置使我拥有了较完善的T型知识体系，既打下了夯实的力学基础，又兼备了控制、xx等方面的背景知识。同时我兼修了多门英语课程，雅思/六级成绩达到…分。本科的学习积累为我在xx方向上的深造研究打下了坚实的基础。

跟几位高赞答主不同，我没有把成绩写上去（我也想写啊！但是分寒碜啊喂，留下悔恨的泪水）。我加权分数87多一些，且加权归加权，我的核心课程分数并不高，如高数上下都为90，结构力学92，流体力学91等，更逞论还有一些不到90的，若是写出来，实在没有威慑力。不能硬碰硬，就只好取巧。因此在学术背景的论述中，我选择避其锋芒，扬长避短，侧重了自己的两方面优势：

  • 课程体系交叉设置 ##我们专业课程设置比较新颖，知识面设计的广度比较大。后来这也成了我在多个面试中的加分项，老师们都很有兴趣
  • 英语突出

找准自己的闪光点是关键

3.科研经历

科研经历是全文的重中之重，也是写起来最让人头大的部分。很多小同学会说我没得过什么有含金量的科创奖项，也没参加过什么项目，素材一穷二白，写不出来怎么办？

别慌，要透过现象看本质。老师让你写科研经历，并不是想看到一堆奖项的罗列、堆砌，而是想通过你的描述，看看你是否具备科研素养、是否有科研潜能，所做的方向与自己是否match。既然如此，我们在叙述中也可以轻结果，重过程。重点讲讲自己在其中，哪些科研能力得到了培养锻炼，如发现问题能力、解决问题能力、查阅文献能力、总结归纳能力、实验设计能力、组织实施能力、科研表达能力，克服困难的毅力和决心等。科研经历比较丰富的同学，可以将这几个能力穿插到多个比赛中描述。科研经历较少的同学可以就一个或两个比赛、项目，进行详述，一网打尽。

例：##摘录自作者 没什么大不了 的回答~

在 2019 美赛中，这是我们第一次参赛，作为队长我组织并规划了赛前的学习准备，赛中的进程控制。在准备期间，我系统学习了线性规划，整数规划，非线性规划，遗传算法等最优化方法，随机模拟，回归分析，多元分析等统计方法，也学习了用 LaTex 工具进行论文写作。在比赛中，我们将选题视为一个多目标多阶段规划问题试图求出最优解，但是由于涉及的变量和过程过多而陷入僵局，之后我将问题简化成分阶段求近似最优解，将救援中心选址类比物流中心选址，再求解集装箱优化问题，最后再求解多无人机多任务分配问题。确定好问题方向后，我们分别查找相关文献并结合问题特点和自身能力寻找可行模型，最终总结归纳出了系统聚类法分区，精确重心法定点，启发式遗传算法求解装箱优化和多无人机协同任务分配模型。我负责分区定点，任务分配的模型建立和编程求解，分别用 Matlab 和 Python 实现了模型并可视化结果。最后便是紧张地用LaTex 进行英文论文的撰写和修改。最终我们获得了二等奖，此次比赛使我的很多能力得到了锻炼，也让我间接感受到了科研的魅力。
此外，在学院老师的指导下我参与了高考志愿系统的开发，查阅相关论文，构建了志愿填报的分档和排序模型。用 python 相关库爬取了汇百智教育、中国教育在线等网站的 31 个省份的历年分数线、一分一档表；整合录取线和一分一档表数据，清除无效数据，将数据MangoDB 转移至 MySql；搜集并整理评价模型涉及到的 MBTI 职业性格测试等心理测试，并从一级学科的层次标记相关专业数据；用 SSH 框架进行 Web 端的开发，采用了主从库读写分离，二级缓存机制等来提高数据库性能。该项目锻炼了我实际编程开发的能力，从需求分析到系统设计到系统实现，几乎经历了一个完整的软件生命周期。在该项目的数学模型构建阶段，我深刻感受到了统计学在数据分析中的重要性，这也是我想在研究生阶段深入攻读大数据统计方向的原因之一。

4.综合素质

我十分看重对自己综合素质的培养，本科期间我积极参加各类活动与学生工作。文体方面，我连续三年担任校xx队成员，并多次代表学校参加比赛，累计获得市级奖项x次。社工方面，我始终践行知行合一，回报社会。累计志愿时长超过100小时，获得xx优秀志愿者称号，率领实践团队深入xx实地调研，并获得校社会实践一等奖。

写综合素质的时候，因为平日里参加的活动多，写起来驾轻就熟，洋洋洒洒就完成了。##不过上面那段，是我把自己的实际情况拿等同条件替换了的## 写综素可以从以下几个方面谈：

  • 文体活动 / 学生工作 /志愿活动 / 社会实践 / 兴趣爱好

对工科生来说，综素在个人自述中不起啥决定性作用，能让老师知道你是个积极阳光好少年就行。但我建议不要过多的表现自己热衷各种活动/工作，点到为止即可，否则老师可能会担心你未来志不在科研，不能沉下心坐住实验室板凳。#所以我就把参加礼仪的事省略啦#

5.未来展望

写这部分的时候，可以登学院官网查看一下都有哪些课题组、每个课题组的研究方向是什么，也可以上知网搜一搜相关老师的文章，并结合自己的科研经历，写出为什么对某个课题组或某个方向感兴趣。可以用这样的句式：

我在大三进行...项目时了解到了...老师的研究方向，并产生了浓厚的兴趣...
在反复学习了...老师在...的论文后，对老师的研究有了更加深入的了解。结合自己的实际工程经历，我认为目前...技术难点还是在于...上，自己有志在这个方向上有所建树，因此更加明确了自己未来的研究方向...
我将阅读大量...相关文献，夯实理论基础，时刻紧跟学术动态...
更多参与进科研项目中，增强自身工程意识...
作为青年人，我生逢其时，也重任在肩...愿为国家的高精尖事业发展贡献自己的绵薄之力



6.结尾

最后，再次感谢您在百忙之中阅读我的个人陈述！

呼！终于终于写完啦！##写到最后都晕晕乎乎了## 希望有帮到各位学弟学妹~

国产手机厂家里，为什么 Vivo 最后一个发布折叠屏？

小蒜苗的回答

从时间来看，vivo的折叠屏的确是一线厂商中最后一个，但从节奏来说，vivo和OPPO还有荣耀其实是类似的，前后差两个多月区别不大。

从整个市场来看，折叠屏的节奏其实分为三个阶段，第一个是华为和三星的行业探索期，第二个是小米MIX Fold的廉价跟进期，第三个是vivo、荣耀和OPPO的成熟扩张期。

从市场和消费者口碑来看，折叠屏整体的成熟过程中，人们对折叠屏产品的期待是越来越高的，对产品缺陷的容忍度也越来越低。

19年三星 Galaxy Fold一代发布的时候，类似外屏体验、内屏折痕、相机素质、系统优化，重量厚度控制这些东西，大家其实都是很宽容的，但到了21年，类似的小米MIX Fold就饱受诟病，这个本质其实也是环境进化带来的结果。

对于vivo来说同样如此，为了追求速度而妥协产品体验是有问题的，对于后发者来说，你有更多的时间来研发，有更成熟的供应链可选择，有更多的行业经验可借鉴，那么同样的，你就必须有更优秀的产品拿出来。

2022年对于折叠屏来说，是一个很重要的时间节点，经过近三年的发展，整个市场对折叠屏的接受度是很高的了，供应链的成熟度和成本控制也都到了一定的阶段，市场即将迎来大爆发。

但同时，行业现有产品各有各的问题，那么对于vivo来说，如何去解决这些问题？能否拿到让人眼前一亮的产品，这个其实才是最重要的。

vivo奉行的是敢为天下后，但作为最后一个入局者，该如何打开局面？

对于折叠屏来说，2022年的产品，必须解决折痕和合不拢的问题，铰链很重要，这个如何解决？

从目前的曝光信息来看，vivo X Fold将会采用苹果供应链厂商提供的铰链，整体成本可能超过1200元，带来的好处则是折痕表现的继续提升。

从曝光的信息来看，vivo X Fold的折痕在正常使用下，已经几乎不可见了。折叠形态下的合不拢问题应该也是完全解决了的。

折叠屏普遍存在的问题是影像能力较弱，这个能否解决？

除了华为Mate X2，业界主流折叠屏产品的影像能力普遍和行业顶级旗舰有差距，对于X系列来说，影像能力是近年来的主打点，那么，vivo能不能在X Fold上延续这一优势？

从曝光信息来看，vivo会采用4800万像素超广角镜头，5000万像素主摄，1200万像素中焦和800万像素长焦，除了主摄采用了GN5，其他部分基本和X70 Pro+保持一致，小道消息，这套影像模组的能力也和X70 Pro+不相上下，甚至部分场景下的表现会更好一些。

如果这一点最终证明是事实，那么vivo X Fold应该是继华为Mate X2之后，第二台搭载了真正意义上旗舰相机的折叠屏手机了，如果确实如此，那么这台机子的期待值会上一个台阶。

受限于体积控制，各家折叠屏的影像系统普遍妥协，但影像能力对于手机的价值已经成为了行业共识，影像能力发展也是各家旗舰机技术能力最重要的体现点之一。当行业普遍妥协的时候，大家考虑的只能是二选一，但这是妥协的结果，不是实际需求的体现。

屏幕部分有两个值得关注的点，一个是形态，一个是素质

曝光信息显示，vivo X Fold采用的是6.53英寸外屏，分辨率FHD+； 8英寸的内屏，分辨率2K+，外屏的长宽比大概是21：9，也就是传统手机的正常尺寸。据说会是两块顶级 120Hz E5 屏幕，且会是内外双120Hz高刷新率，也就是说，vivo X Fold可能会是目前量产折叠屏手机里，屏幕素质最高的机子。

以及，vivo X Fold会是一台大折叠屏手机，在传统手机尺寸下，向大屏延展，当然，这个也很好理解，想要容纳旗舰级的影像系统，机身内部空间要求肯定更高。

不过有一个点，在这样的配置堆料下，整机的厚度和重量控制，包括摄像头模组部分的厚度控制如何，值得期待。

目前曝光的信息是，展开状态下的厚度是6.28mm，如果这点属实，那么vivo X Fold有望成为业界最薄的横向折叠屏手机了。

内屏采用的应该是UTG玻璃面板，能够获得比传统塑料材质面板更高的强度和耐刮性。

外屏使用的是微曲设计，和背板的曲面相呼应，整体的手感会更好。

屏幕指纹，目前行业主流的折叠屏手机普遍采用了侧边指纹识别，这个和业界传统直屏旗舰普遍采用的屏下指纹识别还是有差距的。

从目前的曝光信息来看，vivo X Fold采用的是广域超声波指纹识别技术，且是内屏外屏双屏下指纹设计，这个对于日常体验的提升还是相当明显的。

快充部分，vivo X Fold有可能搭载80W有线+50W无线+10W反向充电，快充规格不够和无线充的缺失也是目前折叠屏手机市场普遍的痛点。

以及，因为折叠屏本身的构造，导致各家在影像之外的一些能力上普遍有妥协，比如屏幕指纹、屏幕素质、快充尤其是无线充等等，包括重量和厚度控制，都相对一般，那么vivo作为最后入局的品牌，这些方面又有哪些建树？

我们简单总结一下vivo X Fold的曝光信息：

铰链：采用苹果供应链提供的铰链，整机的折痕表现会相当出色，合不拢问题会有效解决；

指纹识别：采用广域超声波指纹识别技术，且是内外屏双超声波指纹，折叠屏的指纹识别体验会上一个新台阶；

影像模组：采用X70 Pro+类似影像模组，从而成为折叠屏手机里，少有的真正搭载旗舰影像系统的机子；

屏幕：采用双E5材质高素质屏幕，双120Hz高刷新率，有望成为折叠屏手机产品里的屏幕素质新标杆；

快充：采用80W有线+50W无线快充方案，告别折叠屏手机缺少无线快充的时代；

厚度：展开厚度6.28mm，有望成为行业内最薄的横向折叠屏手机。

如果以上消息都属实，那么vivo X Fold有望成为目前行业内最强大的折叠屏手机，各方面的表现都非常优秀，几乎没有可见短板。

敢为天下后，这个vivo一直在说，但实际上，这个事儿挺难的，先发优势对于消费电子行业是显而易见的，产品节奏靠后，想要获得足够高的市场竞争力，那么就必须在产品能力上领先。

折叠屏手机市场正在快速走向成熟，目前正处在行业爆发的前夕，多个分析机构都给出了类似的预测，2022年折叠屏手机市场将会迎来大爆发。

那么，在这样的时间节点上，vivo作为最后一个发布相关产品的国内主流手机厂商，如果能够拿出最好的产品，那么其市场竞争力，自然不会落下。

中国手机市场是一个被充分教育的市场，人们对于手机的整体认知水平都非常高，这对厂商来说，是挑战，但同时，如果你能够拿出真正优秀的产品，那么，这种挑战也会变成机遇。

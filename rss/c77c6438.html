C 罗告知曼联希望离队，他离队的原因是什么？

海布里de酋长，体育写手/专栏作家/ 好物开箱分享 查看知乎原文

北京时间 7 月 8 日，曼联正式集合备战。一如以往，休战期的备战除了调整竞技状态外，另一大意义在于提升俱乐部的商业收入，因此球队通常愿意将比赛放在海外进行。

依据计划，曼联将先飞往泰国首都曼谷，然后再去到澳大利亚和挪威。令赞助商们失望的是，球队的“票房担当”C 罗却不在滕哈格的大名单内，这无疑会影响比赛的上座率。

曼联官方给出的理由是家庭原因缺席，可明眼人不难看得出，这只是 C 罗和他团队的托词。据部分媒体爆料，C 罗之所以心生去意，是想继续征战欧冠。但这样的说法在我看来不是很能立得住脚，因为红魔无缘欧冠，这件事在五月底就已明朗。C 罗当时不表态，支支吾吾这么久，显然是对高层在夏窗的转会进度不满，进而对新赛季前景缺乏信心。

新任总监穆塔夫对 C 罗的口风格外耐人寻味，从最初态度强硬的“C 罗是俱乐部的非卖品”，到近期假借媒体之口“如果有球队愿意出价 1500 万欧元的转会费，曼联就愿意开启谈判”。曼联官方话术上的改口，与其说是释怀，不如说是迫于无奈。

在巨星话语权空前的“社媒时代”，球星想要改换门庭变得过于容易，更何况他的名字叫克里斯蒂亚诺•罗纳尔多。在 C 罗委婉地向曼联表达去意后，他的经纪人门德斯开始了一系列商业运作。作为足球世界的超级大鳄，门德斯有足够的人脉，让英西法德意一夜之间都写满“C 罗”的名字。

但配合葡萄牙人营销是一回事，实际能将意向转化为报价又是另一回事。从目前转会市场流出的一些消息，那不勒斯和罗马恐无法承受罗的工资。母队葡萄牙体育暂时不在 C 罗的考虑范围内。拜仁方面虽然很欣赏他，但德国班霸这艘航母上似乎并没有 C 罗的位置。

只有切尔西跨出了关键的第一步。美国老板伯利对 C 罗垂涎三尺。即便 36 岁了，C 罗的竞技表现依然称得上出色，上赛季他一共打入了24 粒进球（英超 18 球，联赛第 3），差不多一个人贡献了曼联 1/3 的火力。而球队的二号得分手，B 费只打进了 10 球——这个落差几乎是争冠队和中流队间的差距。

更无肖提，C 罗在世界三大社媒平台上的粉丝总数已超 4.5 亿，为全球名人之最。以商业包装作为球队运营内核的美国老板，自然无法拒绝 C 罗这座“移动的金山”。

可摆在眼前的问题也是显而易见的。这个年纪的 C 罗较为单一的“战术适配性”未必能适应图赫尔强调高压、侵略性与机动性的 3-4-3 阵型。Opta 的数据甚至显示，在上赛季欧洲五大联赛出场时间达最低标准的所有前锋中，C 罗是参与前场紧逼次数最少的球员。你很难说，朗尼克在曼联的失败与罗没有某种微妙的内在联系。

现如今，无论是德系教练还是英系教练，高位逼抢几乎是一支球队里不可或缺的战术配件。豪门更是如此，这无疑限制了罗下一站的选择性。

在真空环境下，如果 C 罗去到一支类似多特蒙德或者莱比锡红牛这样的球队。在德甲大开大合，较为开放式的环境下，他依旧顶级的无球终结能力是可以弥补跑动不足的欠缺。而年轻化的球队也需要这样一个老大哥来把舵，如果 C 罗愿意降薪，愿意在一些不太重要的比赛中打替补，那他很有可能会成为伊布之于米兰城那样的“传奇”。可那也就不再是 C 罗了……

也许 C 罗不那么偏执，那么执拗，没有那么病态的好胜心的话，他可以更聪明更讨喜。但更大的可能性是，他会泯然众人，也远远达不到今天的成就。我们不能在评价 C 罗时陷入“二元思维割裂”，如果你喜欢 C 罗的话，你就需要接受他的全部。

然而，这对于路人球迷——特别是经历“梅罗时代”的球迷们来说，这样的结果还是会稍显遗憾。

正如阿瑟 @segelas 所言，C 罗缺乏的不是荣誉，而是传奇故事的圆满。罗今日一系列的争议所为，会让后人在评价这个竞技水准无比卓越的个体时变得更加困难。

爱他执念者，会将他晚年的混乱视为自毁的悲剧宿命，如痴如醉。恶他自私者，则会把他打造为一个只有自我的团体性运动的反面，弃若敝履。

ONLYOFFICE 桌面编辑器 9.0 版本已发布：更多文件支持、AI 增强、图表查看器等更新

继 ONLYOFFICE 文档 9.0 版本重磅发布之后，桌面办公套件也迎来了适用于 Windows、Linux 和 macOS 的新版本。ONLYOFFICE 桌面编辑器 9.0 版本不仅继承了自托管版本主要的新功能和改进，还进行了一些独特的更新。阅读本文，了解详情。

让我们先快速回顾一下新版在线版本的主要功能，这些功能同样适用于 ONLYOFFICE 桌面编辑器 9.0 版本。

文档、表格、幻灯片和 PDF 全新界面

在 9.0 版本中，您可以切换到编辑器新添加的浅色和深色主题，使用全新主题界面。这些新界面中，UI 元素与图标均经过重新设计，界面更加清晰易用，提供现代直观的用户体验。
位置：视图选项卡 -> 界面主题 -> 浅色/深色主题

除了编辑器界面全面升级外，桌面应用的起始窗口也重新进行了设计。简洁的设计让您可以更快找到所需功能，避免被无关内容干扰。

全新图表查看器

9.0 版本引入了 ONLYOFFICE 图表查看器，使套件功能大幅拓展。您可以在各种平台和设备上打开和查看图表，包括电脑和笔记本。

新增文件格式支持

除了 ONLYOFFICE 图表查看器最新版所支持的 .vsdx 文件，9.0 版本还支持以下文件格式：

  • Markdown 文件（.md）
  • OpenDocument 图形文件（.odg）
  • .xlsb 文件（无需预先转换即可编辑）

重磅 AI 功能增强

除了界面与格式支持方面的提升，9.0 版本还集成了实用的 AI 工具，以帮助提高您的工作效率：

  • 使用 OCR 功能从扫描 PDF 文件中提取文本；
  • 使用表格新 AI 功能，在表格中智能应用公式或进行复杂数据分析；
  • 借助 AI，使用指令创建新的 ONLYOFFICE 宏，并快速转换 VBA 宏。

PDF 编辑器的更多功能

ONLYOFFICE PDF 编辑器在 9.0 版本中功能增强，支持拖放 PDF 页面实现快速重排，还可使用 Ctrl + C / Ctrl + V 快速复制页面。

从右至左支持&乌尔都语本地化改进

ONLYOFFICE 办公套件最新版本中 RTL（从右到左）语言支持进一步增强：

  • 表格中新增 RTL 支持；
  • 改进了文档中 RTL 文本与段落的使用体验；
  • 幻灯片与 PDF 支持从右到左/ 从左到右文本方向切换。

此外，9.0 版本还新增乌尔都语界面，同时也为其他语言带来细节优化，例如：简体中文界面下可设置默认字体大小，数字可使用印地语中的数字显示。

更多功能

以下是 9.0 版本中其他功能的简要概览：

  • 文档中段落边框可通过工具栏按钮快速设置，内容控件功能增强；
  • 表格支持异步计算；
  • 幻灯片中可动态呈现文本动画；
  • 各编辑器中的数据可视化与图表自定义能力增强。

想详细了解新功能并掌握高效使用方法，请阅读此篇文章或预约观看6月25日的网络研讨会。

Linux 平台新增拼写语言检测功能

在 9.0 版本中，Linux 用户可在“设置”中启用或禁用“拼写语言检测”选项，能够改善频繁切换键盘语言时的输入体验。

无需调用系统对话框即可打印

ONLYOFFICE 桌面编辑器 9.0 版本支持通过文件选项卡的打印选项，或或顶部工具栏中的打印文件选项直接打印文件，无需打开系统打印对话框。如仍需使用系统打印窗口，可在设置中启用相关选项。

.exe 安装中的文件关联功能

在 Windows 10 或 Windows 更高版本中安装桌面应用时，.exe 安装程序允许你选择要与 ONLYOFFICE 关联的文件格式。你可以快速选择所有可用选项，或手动勾选所需格式。

获取 ONLYOFFICE 桌面编辑器 9.0 版本

最新版的 ONLYOFFICE 桌面编辑器现已支持 Windows、Linux 和 macOS 及 Flatpak，可在官网获取。Snap 和 AppImage 将稍后上线，敬请关注！

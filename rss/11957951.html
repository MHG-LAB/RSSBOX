如何评价 4 月 11 日发布的 Vivo X Fold 折叠屏真旗舰？有什么亮点？

你的阿笑笑的回答

前言

2022年的今天，在折叠屏手机这一赛道上，各家的折叠屏产品都已经纷纷开始起跑，而近期vivo也是宣发了自家的折叠屏旗舰。作为折叠屏手机领域的后来者，vivo的首款高端商务折叠屏旗舰，vivo X Fold要面临的是折叠屏前辈们给予的巨大压力，想要冲出重围，vivo X Fold自身的产品力显得尤为重要。

下面我们就一起来回顾一下，vivo X Fold的具体配置。

屏幕

说到折叠屏这种产品，屏幕肯定是我们必须要关注的重点。而在屏幕这一块上，又有两个部分是我们需要关注的重点：一个是屏幕的折痕，也就是折叠屏铰链部分的处理；另一个则是屏幕的素质，具体一点就是屏幕的尺寸、刷新率、色准等。

铰链

我们先来看铰链部分。这次vivo X Fold在铰链的零部件中采用了液态金属锆合金、F53航空高强度钢、手撕钢片等六重航空材料。为了解决折痕问题，vivo在铰链的中央做了一条一体式锆合金龙骨，作为整条铰链的“骨架”，为铰链提供支撑。而在龙骨之上，又做了一条一体式锆合金浮动中板，当合起内屏时，浮动中板下降，给屏幕留出折叠空间；而当内屏打开时，浮动中板上升并托起屏幕，屏幕得到一个平整的支撑面，也就能够变得更加平整。

其次，在屏幕的材质上，vivo X Fold选择了UTG超韧玻璃，可以减轻屏幕的塑性变形，从而提升屏幕的耐用性。同时UTG超韧玻璃能够允许更小的弯折半径，这使得vivo X Fold做到了2.3mm的弯折半径，被弯折区域变得更小，折痕也能够变得更窄更浅。

屏幕素质

说完了铰链，我们再来看屏幕方面。这次vivo X Fold配备了8.03英寸的内屏，尺寸相当于一台小型的平板电脑，不论是商务办公还是影音娱乐都有着更好的体验。同时内屏的分辨率为2160*1916，屏幕的精细度上也并没有因为屏幕太大而降低。而在外屏方面，vivo X Fold配备了一块6.53英寸的微曲面外屏，分辨率为2520*1080，长宽比达到了21:9，这样的尺寸更加符合普通用户的使用习惯，在外屏的日常使用中也不会受到太多的影响。

不同于普通手机，折叠屏手机在使用的过程中会频繁切换内外屏幕，因此两块屏幕的显示效果要尽可能做到一致，才不会让用户在使用时出现明显的割裂感。因此vivo X Fold采用了E5发光材质，三星钻石排列，同时出厂时便对两块屏幕进行色彩校准，保证两块屏幕尽可能拥有一致的显示色彩。

同时在屏幕刷新率方面，vivo X Fold的内外屏都做到了120Hz的刷新率，这是让我比较惊讶的一点。因为目前做到内外屏双120Hz刷新率的就只有三星，其他家基本上都是90+120或者60+120的方案。综合vivo X Fold所有的屏幕配置，我认为在折叠屏领域中，vivo X Fold绝对是数一数二的存在。

3D超声波双屏下指纹方案

这次vivo X Fold并没有选择传统折叠屏产品使用的侧边指纹，而是选择为内外双屏都配备上超声波屏下指纹，这使得vivo X Fold成为了目前行业内首款搭载超声波屏下指纹的折叠屏手机。

不得不说，vivo X Fold会采用双屏下指纹方案，我真的是很惊讶的。因为按常理来说，双屏下指纹的方案应该会占用更多的手机内部空间，这对于折叠屏手机来说是不太划算的，而vivo却选择了逆流而上，我很好奇是什么驱动着vivo选择了这套方案，又是怎么解决这套方案中出现的难题。

性能

作为2022年的旗舰产品，vivo X Fold在性能方面的配置完全是毋庸置疑的。

在硬件方面，高通骁龙8 Gen1（SPU定制）+超频版UFS 3.1+增强版LPDDR5性能三件套直接拉满，同时支持内存融合2.0，可以将12GB运行内存扩展到等效16GB使用。另外vivo X Fold还配备自研的三重全时域性能引擎，优化系统流畅度、复杂网络场景以及日常续航。

音质一直都是vivo的强项，因此vivo X Fold上也自然不会缺席。vivo X Fold搭载了一颗独立的CS43131 Hi-Fi芯片，让用户在音乐以及视频场景中都能享受到Hi-Fi带来的顶级听感。此外，vivo X Fold配备了大音腔对称式立体声双扬声器，给用户提供更好的外放体验。

续航方面，vivo X Fold配备了4600mAh电池，同时支持66W有线闪充、50W无线闪充以及10W的反向充电。另外考虑到商务人群在出差时会携带多台电子设备出门，因此vivo X Fold标配了1.5米数据线以及双Type-C口氮化镓充电器，实实在在地解决了出差充电时的需求，这一点我觉得必须要给vivo一个好评。

影像

影像，是近几年来vivo主要的发力点。此前vivo就已经与蔡司达成战略合作关系，推出过不少优秀的影像旗舰，因此在vivo X Fold上也同样不会缺席。vivo X Fold全系列搭载蔡司光学镜头，同时配备蔡司T*镀膜，镜头的综合光学性能以及抗眩光能力非常不错。

虽说折叠屏手机的内部空间有限，留给影像模组的并不多，但vivo X Fold显然也没太将就，就焦段覆盖上同样满足了大部分的日常使用焦段。主摄传感器采用了5000万像素的三星GN5，超广角镜头传感器则是采用了4800万像素的索尼IMX598，二倍人像镜头传感器采用了1200万像素的索尼IMX663，5倍潜望式长焦镜头则是采用了800万像素的OV08A10。

这套影像模组让vivo X Fold能够轻松应对各种场景，同时得益于折叠屏手机双屏的设计，还能通过外屏或者内部半屏进行取景，极大丰富了vivo X Fold的摄影体验。

软件适配

软件适配，是折叠屏手机必不可少的一部分。想要充分发挥出vivo X Fold的特性，光有硬件实力往往是不够的，还需要有软件的配合。

先来说说多角度悬停，这是我觉得很可爱的一点。vivo给X Fold做了一个悬停的范围，大约在60-120度内都可以悬停，而借助这个形态，vivo也是给X Fold整了一点小功能。例如在拍照方面，我们就可以利用悬停形态代替三脚架进行延时摄影或者自拍；或者是看视频的时候找不到手机支架，那么就可以直接将手机折起来，利用上半屏来播放视频；又或者是在阅读邮件、电子书，需要输入文字的时候，也同样可以将X Fold悬停在合适的角度，悬停状态下输入法恰好基本能够占半屏，也就可以像笔记本电脑一样敲敲敲了。

多任务处理也是折叠屏手机很重要的特性。vivo X Fold支持了应用分屏，三指上滑即可启动，能够将两个不同的应用同时显示在大屏上，还可以将常用的分屏组合收藏至桌面，下次轻点即可开启。其次是应用间的协作，只需要将窗口内的图片、视频等文件长按拖拽到另一窗口，即可完成粘贴或是发送。然后是应用分身，vivo X Fold能够在大屏上打开2个相同的应用，操作互不干扰，便于进行内容对比、多信息同时浏览等。最后，vivo X Fold还支持了分屏小窗同时启动、切换和协作，在分屏界面上也可以调出小窗，从而在大屏上更高效地处理多任务。

作为一款商务旗舰，vivo X Fold在办公场景中也做了一些特别的适配。比如琥珀扫描这个我觉得真的很有用，可以扫描文档，提取文字，识别表格，转为PPT，又或者是直接扫描出来进行打印，我想这些功能对于办公场景的作用，经历过的人应该都懂吧。

然后是跨设备办公。其实很多手机都具有多屏协同的能力，但往往大家仅限于同一品牌的设备，因此往往也受到不小的限制。而vivo X Fold则是不讲武德，不但要适配Windows电脑，甚至连Mac也不放过，通过搜索、扫码、数据线等方式实现将手机与电脑进行互联与同步，直接解决手机与PC端信息共享的痛点。

另外两款新品

作为vivo首款折叠屏手机，vivo X Fold的完成度很高，相较于其他折叠屏产品也是展现出很强的竞争力。

除了vivo X Fold之外，这次还同时发布了vivo X Note旗舰商务手机以及vivo Pad平板电脑，这里我们也进行简单的介绍。

vivo X Note

vivo X Note采用了一块2K分辨率E5材质的120Hz高刷屏，支持LTPO自适应刷新，康宁大猩猩GG7玻璃盖板，21:10的机身比例。性能方面，vivo X Note同样配备了骁龙8 Gen1旗舰芯片+超频版UFS 3.1+增强版LPDDR5性能三件套，同时配备了自研V1影像芯片，加强了手机的影像处理能力。续航方面，vivo X Note配备了一块5000mAh大电池，同时支持80w的有线闪充、50w的无线闪充以及10w的反向充电，配置直接拉满。

值得注意的是，vivo X Note搭载了大面积的超声波指纹识别模块，识别区域高达578mm²。

vivo Pad

在平板电脑这一领域，vivo也是后来者。而作为vivo首款平板电脑产品，vivo Pad被定义为vivo旗舰级的平板电脑。

vivo Pad采用了一块11英寸的2.5k 120Hz原色屏，支持HDR10、杜比视界等，观感体验直接拉满。性能上，vivo Pad选择了骁龙870移动平台，同时给其配备了13626.6 mm²的石墨散热片，保证平板在高负载状态下的散热表现。另外vivo Pad还配备了四声道的立体扬声器，自适应杜比全景声，搭配它2.5k的屏幕，影音效果直接拉满。续航方面，vivo Pad配备了一块8040mAh的大电池，同时配备44w有线闪充，大约一个半小时充满电，满电状态下可连续看剧11.5个小时，连续移动办公15个小时。

对于安卓阵营的平板电脑而言，硬件其实是次要的，平板的生态才是影响使用体验的关键要素。vivo Pad打通了多设备壁垒，实现手机与平板的深度互联，实现了NFC一碰互传、内容跨屏同步以及任务跨屏接力等功能。而在多应用场景下，vivo Pad也支持应用多窗以及应用分屏的功能，保证了用户在日常应用上的体验。同时vivo还为平板准备了一套智能触控键盘以及触控笔，键盘采用了全玻璃面板，同时配备了一个超大的触控板，并且为触控板适配了一套丰富的触控交互方式，大幅度提高键盘的效率。另外vivo Pad的触控笔除了支持4096级压感，还支持了息屏速记、手写识别、智能按键等功能。不论是日常学习还是商务办公，vivo Pad都能很好的满足用户的需求。

这就是发布会的三款新品，每一款产品都有着很独特的亮点，我很期待它们上市之后的表现。

如何评价华为 4 月 28 日发布的 WATCH GT 3 Pro 系列手表？有何突出的亮点和槽点?

綾音Ayane的回答

今天的发布会上，华为 WATCH GT3 Pro系列真的是赚足了眼球。在这之前，我印象中的智能手表发布会都是先讲男款，顺便讲一下有女款。而这次华为 WATCH GT3 Pro陶瓷版一出场，我所在的各种数码相关的群就仿佛炸了一样。

不得不说，这块手表实在是太好看了。

而非常荣幸的是，我又提前体验到了华为 WATCH GT3 Pro陶瓷版。也能够在今天给大家分享一下我的体验。

我手里这块是华为 WATCH GT3 Pro陶瓷版，皮质表带的版本。

迫不及待先放一张上手图

1.外观

很长一段时间以来，陶瓷表在小姑娘中间就有着极高的人气。比如说著名的小香J12，又比如说最近炒疯了的Swatch陶瓷表。好像陶瓷表从妈妈们年轻的时候到现在都是各类搭配里面最常见的单品之一。


那么，你有想过，有一天能拥有一块陶瓷智能手表吗？

华为WATCH GT 3 Pro陶瓷版就作为一个辣么大的惊喜出现在我眼前。

华为WATCH GT 3 Pro陶瓷版整体表盘为白色，表盘周围有银色的圈圈设计，旋钮的边缘和ECG也是银色，整体设计细腻又温润。

表盘尺寸为43mm，女孩子戴在手上会显得格外秀气。

表带有真皮和陶瓷表带可选。我手上的这一块是真皮表带。表带颜色为乳白色。表盘与表带衔接处并不突兀。和前几代手表一样，表带的拆卸也非常容易。


因为整体上没有过于突兀的颜色和形状，让这块手表几乎可以搭配所有颜色和风格的衣服。

甚至和我早市上20块买的睡衣搭配也毫无违和感


（忽略早晨做饭溅在衣服上的油点子）

此处悄悄跑个题，我感觉Watch GT 3 Pro的乳白色真皮表带我感觉换到Watch GT 3 42mm上也会很好看。

因为表盘是陶瓷材质，即使是在运动之后也不会觉得黏糊糊的，更不怕划。这一点大大的好评。

而陶瓷表带的Watch GT 3 Pro则是这个样子的（图片来自发布会截图）

2. 充电和续航

考虑再三，这次依然把充电和续航放在靠前的位置。

在续航方面，华为Watch GT 3 Pro 陶瓷版和之前测评过的华为Watch GT 3 42mm基本持平。

官方数据是典型场景续航可以达到7天，重度场景续航可以达到4天。

因为测试时间比较短，我得到的数据是在开启几乎除了息屏显示以外所有功能的情况下，续航大约为5天。和官方数据基本持平。

但和上一代不同的是，HUAWEI WATCH GT 3 Pro充电速度提升了30%，仅需10分钟即可充电25% 在测试的时候我走了个神。于是得到的数据是28min充了36%

比较值得一提的是，HUAWEI WATCH GT 3 Pro的无线充电底座没有金属触点，这也减少了金属过敏人群皮肤过敏的可能性，以及……我老爹天天疑神疑鬼的觉得金属触点会不会生锈。（虽然事实证明，正常使用并不会生锈。）

3. 自由潜水和防水

原本以我的习惯，这里应该写UI或者运动的。但由于HUAWEI WATCH GT 3 Pro在之前预热阶段，就因为它的防水性能而引起了一大波讨论。咱们就先来聊这部分。

HUAWEI WATCH GT 3 Pro系列配备了专业的自由潜运动模式，它可以支持最大深度为30米的自由潜运动。而在自由潜运动中，我们这些业余爱好者的天花板也就是aow了，而当你持有aow，你最大可潜深度也是30m。所以这个数值可以覆盖绝大部分自由潜爱好者。

HUAWEI WATCH GT 3 Pro的自由潜运动模式中呈现的专业数据包括下潜上升速度、实时深度、最大深度、潜水时间、水面休息时间、下潜次数等；支持水面卫星定位，可精准定位入水前位置；具有潜水告警提醒功能，用户可自行设置深度提醒与告警、时间提醒与告警等。

虽然现在没有机会去潜水，但我们依然可以用HUAWEI WATCH GT 3 Pro进行闭气训练

当然……其实对于大部分人来说，可能很少会用到这个功能。但咱们会觉得，HUAWEI WATCH GT 3 Pro系列的卖点之一就是自由潜，防水性能想必不会差吧。

而对于数码产品，尤其是智能穿戴设备，不知道大家是不是和我一样，IP68防水我不会戴着洗澡，5ATM我不会戴着游泳。但理论上，IP68级防水的智能穿戴设备就完全可以戴着游泳了。

我总会觉得它是个数码产品，虽然它防水，但我还是小心点儿用吧。

所以对于我来说，华为WATCH GT 3 Pro大概会更多的出现在游泳池，浴缸，温泉，以及各种各样的水上项目里。


当然，也因为表盘本身防水性能好，而真皮的特性之一是不防水。从这一角度来说，如果你会习惯戴着它游泳和潜水，可以考虑选择陶瓷表带的版本。

4. 运动

下一个要聊的就是运动了。

此处我特别想吐槽一句，能忽悠我去跑步的只有智能手表……

之前某个小哥哥忽悠我跟他一起跑步，忽悠了好久，都不如华为Watch GT 3的召唤。而这次有了华为WATCH GT 3 Pro陶瓷版，咱们不出去跑一圈儿好像也不太合适。

在跑步的过程中，我们可以随时看到各种各样的数据，比如说配速、心率、跑步距离，也可以看到智能评估跑力指数和训练压力，及阶段训练负荷、恢复程度等。


这里我们提一下华为WATCH GT 3 Pro的音乐播放功能，它不仅能够利用蓝牙控制手机播放音乐，也可以独立播放音乐。

这里有个非常贴心的设计，在运动过程中，我们只需要左右滑动手表，就可以调出播放音乐的界面。

华为的智能跑步计划，还可以根据用户个人身体参数和跑步历史数据，训练目的，为用户指定训练计划。

虽然这个计划对我来说……基本上就是拿来咕咕咕的（小声）

当然还有LT，也就是我们之前提到的跑力指数功能，我们可以选择LT图标，然后进行跑力指数测试。这里我要提醒大家一下，这个测试非常非常非常累……所以一定要在第二天没啥事儿的情况下进行测试

除了跑步之外，它还支持100多种运动。我用的比较多的是瑜伽，平常跟着华为运动健康app里面的瑜伽教程做一做瑜伽，然后就听见自己身上嘎巴嘎巴的，果然还是运动太少了。

剩下比较值得一提的就是前面提到过的自由潜运动模式，和高尔夫模式了。

华为WATCH GT 3 Pro在高尔夫模式中增加了球场模式。它搭载了国内300+全新的球场地图，提供了完整的球道、果岭和障碍区分布的视图，让用户能够轻松掌握球场全貌及功能区构成。根据精准的GPS定位能力能够实时展示用户当前位置到达果岭前沿、中心和后沿的距离。为用户致胜果岭提供有力的保障

当然像我这种只能在练习场挥两杆子的菜鸟，基本上只能用到高尔夫练习场模式。如图所见，华为WATCH GT 3 Pro可以记录挥杆速度、挥杆节奏、上杆时间和下杆时间等专业数据。

不过值得一提的是，挥空杆是不被记录的。我偷懒试图用推杆当长球杆用，竟然也不能识别QAQ

5.户外安全

刚聊玩运动，我觉得我们有必要再来聊聊户外安全。

华为WATCH GT 3 Pro系列内置海拔气压计，可以通过户外的气压变化判断恶劣天气，如遇气压急剧变化，就能在恶劣天气来临前进行温提醒。

表盘可显示每天日出日落的时间，支持月相呈现，还可通过表盘实时了解潮汐变化

6. 健康

许多人购买智能手表的理由之一，就是可以随时看到自己的健康状态，还可以随时监测家庭成员的健康状态。

我们先简单晒一下我的日常健康数据吧。因为这两天绫音喵处于生病状态，数据看起来也不是很正常。

心脏健康研究

除此之外，华为WATCH GT 3 Pro依然可以用心脏健康研究APP，我们可以很方便的看到自己的心率异常状况和睡眠呼吸暂停状况。


因为之前我一直有佩戴华为Watch GT 3，而呼吸暂停从来没有过问题，所以这次也没有开启这个功能

血管健康研究

另一个我常用的健康相关的app就是血管健康研究，我们只需要在佩戴好手表的情况下，把手指按放在ECG按钮上，就可以得到一个动脉弹性程度的筛查结果。

我们甚至还可以查看波形（虽然我也看不懂）

华为WATCH GT 3 Pro还可以单独测量心电图。我们只需要坐好，把手指放在ECG按钮上，等待30s即可在华为运动健康app-心脏健康看到本次测量的心电图。

7. 其他

除了上述功能之外，还有一些我经常用到的功能。

比如说语音助手。有时候懒得把手机拿起来，就用手表呼唤小艺，问问几点了，定个闹钟，或者问问谁是最可爱的人。

再比如说天气，每天起床都可以抬腕看看天气真的是太棒了，有时候还能在没起床的时候瞟一眼室友穿的什么衣服，告诉他今天该加衣服还是减衣服。

在之前使用华为Watch GT 3的时候，我还会用手表进行导航。尤其是之前和基友逛五大道，手里又是奶茶又是零食的，实在是懒得把手机掏出来。而用手表导航就不会占用本不富裕的手（x）。喝着奶茶看着导航完全不是问题。

再比如说手电筒。虽然华为Watch GT 3 Pro的手电筒只是亮度调到最大，屏幕显示为纯白色。但是在夜里上厕所的时候非常有效——在家里我夜里起来都是直接喊小艺，只有在宿舍的时候我会用这个功能照亮我爬下来的路。据我已婚的表姐说，语音助手会把表姐夫和小外甥都吵醒，手表自带的手电筒在这时候也非常有用。总的来说是个相当温馨的功能

还有类似于找手机啊，NFC啊，遥控拍照等等，日常生活中也非常有用。这里有个比较“缺德”的玩法。用华为Watch GT 3 Pro可以直接操作打开手机的相机。有时候和表兄弟姐妹出去吃饭，我总是就这么不知不觉的拍好多他们的表情包，然后在我们家族群里面各种传播开。

8.售价

在前些天刚拿到华为Watch GT 3 Pro陶瓷版的时候，我就特别特别特别想给大家安利它了。

作为一个颜狗，同时还是一个懒蛋，一块几乎能搭配我所有衣服的智能手表真的是刚需。

除了颜值，它第二吸引我的就是它的“抗造”，连海水都不怕的手表，我得多折腾才能把它也折腾坏。

而这次体验过程中，最让人难过的就是绫音喵在生病，于是母上大人死活都不让我泡澡。我只能单独把华为Watch GT 3 Pro陶瓷版丢进浴缸里，也算让它替我泡个澡吧。

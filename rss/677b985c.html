中考高考临近，考生怎么吃更有利于身心健康？

中考，高考临近，考生怎么吃更有利于身心健康？
章沪川，一个懂得用火的文明人｜公众号：小川食堂 查看知乎原文

我高中老师前几天突然问我，家里小朋友马上要中考了，这段时间应该吃点什么好。

然后我就陷入了感慨，当年上高中的时候还是个小不点，竟然一下子马上就要中考了。

参加工作后很长一段时间感觉时间似乎停滞掉了。果真只有看到小朋友长大，才一下子发觉时间过的飞快。

跑题了跑题了，想说中考高考前，家长对孩子的饮食可能会感到有些许压力。因为在学习上好像也做不了什么，只能在生活上对孩子尽可能的给予帮助。

我老师就说孩子早餐和周末回来都不知道该给他吃什么，每次准备食物都有一种“横冲直撞”的焦虑，逮着什么就做什么，没有一点计划性。

其实关于考生的考前饮食，家长不用过度担忧，总体的应对原则是，平时吃什么现在也吃什么就行了。

我觉得在大考前，有一样东西更重要，那就是心态。

我有一高中同学，前不久以第一作者之名发了篇 nature，他自己非常谦虚，发了个朋友圈就继续默默做实验去了，但我们所有同学都激动万分，到处夸耀“我的科学家同学“事迹（比如我 hh）。

他在上高中时就有一个非常明显的特点，心态巨好，特别淡定。

不管是每次模拟考还是最后的高考，他都非常“佛系“，保持自己的学习节奏，从来不和我们“卷“。

他就是凭借着他良好的心态、踏实的努力、以及认真钻研的精神，一步一个脚印，从刚上高中时的一两百名，到高考顺利考上浙大，然后顺利读研读博，到今天顺利发 nature。

说了这么多，想跟各位家长考生表达的是，在考试前的时候放宽心，保持好心态就可以啦。顺其自然，一切自有最好的安排。

当然也有一些能帮助保持身心健康的饮食大建议，各位家长可以在不改变以往饮食习惯的基础上加以灵活运用。

1、早餐质量要足够高，才能保证上午三小时充足精力，蛋白质 + 坚果可以延缓午前出现饥饿感。如果早上吃不下太多，可以带一瓶酸奶在上午迟点的时候喝，或者带一包每日坚果当零食吃。

在考前的时间（其实平时也一样），家长要特别重视早餐的质量。我自己在上学时有一点其实做的很不好，就是不爱吃早餐。经常吃两片面包或者吃一碗拌面就过去了，早上时常没有精神，困困的，永远是一副刚睡醒的样子。

然而在我工作后开始关注营养，关注早餐质量开始，我发现良好的早餐能让我一整个上午都保持精力充沛，思维活跃。

就.....感觉有点亏.....

我那时要是好好吃早餐，说不定现在也有发 nature 的同学一半厉害呢（开个玩笑哈哈），

早餐最好是有一定的碳水 +2 样蛋白质 + 蔬菜（或者水果）+ 坚果。

给大家举三个例子。

 早餐
第一天 黑芝麻豆浆 1 杯 + 香菇菜包 2 个 + 煮鸡蛋 1 个
第二天 抱蛋煎饺 + 牛奶 1 杯 + 一个小苹果
第三天 香甜八宝粥 + 煎鸡蛋 1 个 + 蓝莓坚果酸奶 1 小杯

黑芝麻豆浆的做法：黑豆 12g、黑米 13g、黑芝麻 5g、核桃 10g、小米 5g、水 500ml，用豆浆机或破壁机的豆浆模式，打完之后加 250ml 牛奶和 20g 冰糖，搅拌两分钟。

这里是 3 杯分量，很好喝，我经常做这个。

抱蛋煎饺做法：平底锅加一点油和水，加入饺子盖上锅盖中火煎到饺子底部出现脆皮，然后淋入蛋液煎熟（饺子就用家里平时吃的饺子，馅料是荤素搭配的就成，例如韭菜猪肉馅，芹菜猪肉馅）。

八宝粥做法：紫米，糯米，小米，花生，红枣，桂圆肉，葡萄干，莲子，百合，加 8 倍水煮粥，电饭锅预约（但不要加太多糖）。

蓝莓坚果酸奶：酸奶里拌点蓝莓和坚果。

2、午餐要控制精白主食，避免下午昏昏欲睡，影响大脑工作效率。

吃点粗粮的好处我已经讲过很多次了，但这事儿不好办，因为孩子可能平时没有吃粗粮的习惯。怕孩子抗拒甚至出现肠胃不适。

可以用我之前出过的一个藜麦小米饭食谱，口感上和白米饭非常接近，不会让孩子难以接受，而且也不容易引起胃胀。少量粗粮还改善了白米饭的营养。

长期用杂粮代替米饭会对身体有何影响？

在考前比较早的时候就可以让孩子试吃了，但如果发现孩子还是不喜欢，就改回白米饭。

做法：

大米 100g，小米 15g，三色藜麦 15g，玉米 15g，豌豆 15g

大米小米藜麦淘洗好后和平时煮白米饭的程序一样，玉米粒和豌豆粒用水煮一下最后一起混合。

同样给各位家长举三个例子。

 午餐
第一天 红烧茄子 + 鲫鱼豆腐汤 + 蒜蓉空心菜 + 藜麦小米饭
第二天 猪肝菠菜汤 + 红烧鸡翅 + 胡萝卜炒莴笋 + 藜麦小米饭
第三天 清炒包菜 + 银鱼水蒸蛋 + 番茄炖牛腩 + 藜麦小米饭

3、晚餐不要吃的太油腻，以免影响睡眠。

晚餐宜清淡一些，吃太油腻会影响消化和睡眠。

但清淡不是寡淡，也要把饭做的让孩子有胃口吃下去哦。

以前我妈有次不知听谁说考前应该吃清淡点，然后就把饭菜做的很“清淡“，清淡的我都不想吃了。

举三个晚餐的例子。

 晚餐
第一天 玉米萝卜排骨汤 + 西芹炒肉丝 + 香菇烧油菜 + 红薯饭
第二天 茄汁虾仁烧豆腐 + 上汤西兰花 + 红豆汤 + 蒸紫薯一小块
第三天 白切鸡 + 丝瓜蛋汤 + 白灼菜心 + 小米粥

茄汁虾仁烧豆腐做法：2 个熟透的番茄切成小块，下油锅炒出汁，然后加点水，加一个碾碎的咸蛋黄，加点生抽、蚝油和盐。虾仁加点盐、白胡椒粉、少量白糖先腌一下然后下锅煎一煎，老豆腐切成块，加点盐稍微煎一煎，然后和虾仁一起放茄汁里稍微煮一会，拌匀后出锅

上汤西兰花：准备少量肉片，1-2 个水煮蛋（切小丁），3 瓣蒜，一点平菇撕成小条待用，西兰花切块。先下锅炒香肉片和蒜，然后加水煮蛋炒一炒，加点料酒，再加半碗水煮开，用盐和白胡椒粉调味，加平菇和西兰花再煮 2 分钟，出锅撒一点枸杞和葱花（西兰花比较难熟，可以用水先煮熟，没买到西兰花的花可改成娃娃菜菠菜之类的）。

还有其他的一些建议：

4、大豆、鸡蛋（黄）富含卵磷脂，卵磷脂是神经递质乙酰胆碱的原料物质，对维持记忆力有好处。

5、奶类、绿叶蔬菜和杂粮，对考生有额外的好处。牛奶、酸奶中含有大量钙，以及牛奶蛋白分解产生的安神肽，能够帮助维持考生情绪稳定，绿叶蔬菜中的镁也有类似的作用。杂粮中的 b 族维生素对大脑高效工作十分有帮助。

6、少量多次喝水，有助于维持身体活动和认知能力

7、不要喝冷饮、饮料、吃烧烤，每天可以适当吃一点儿水果，但是蔬菜吃够了的话水果不爱吃也没事。

差不多就是这些了。

我是章沪川，提前祝各位考生考试顺利，考上理想的学校。

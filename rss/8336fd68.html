ChatGPT 登陆 Windows

OpenAI 宣布已开始预览其 AI 聊天机器人平台 ChatGPT 的专用 Windows 应用程序。

下载地址：https://openai.com/chatgpt/download/

OpenAI 表示，该应用目前仅供 ChatGPT Plus、Team、Enterprise 和 Edu 用户使用，是一个早期版本，预计在今年晚些时候推出“全面体验”。

“通过官方 ChatGPT 桌面应用程序，你可以就文件和照片进行聊天。这款应用为你带来了 OpenAI 最新的模型改进，包括访问我们最新、最智能的模型 OpenAI o1-preview。”

Windows 版 ChatGPT 应用程序可在大多数 Windows 10 机器上运行，但目前与其他 ChatGPT 客户端相比存在一定的局限性。它尚不支持语音（包括高级语音模式），而且与 OpenAI 的 GPT Store 的某些集成功能也不支持。

与 MacOS 版的 ChatGPT 应用程序一样，Windows 版的 ChatGPT 应用程序可以让你在工作时将其最小化为一个小的“companion”窗口，与其他应用程序并列。你可以向它上传文件和照片，让它汇总文档，并通过 OpenAI 的 DALL-E 3 图像生成器创建图像。

如何系统地自学 Python？

李金ECHO的回答

Python 3版本的笔记已经安排上了，最近会逐步放github上。

基于这个笔记的实体书了解一下？书名叫《自学Python——编程基础科学计算及数据分析》，将笔记的内容进行了系统整理，并补充了一些基础知识，可以对照两边的目录看差异。基于2.7。京东有自营。

Python 其实挺简单的，也挺强大的。

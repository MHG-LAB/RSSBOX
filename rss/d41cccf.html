人们常说的电影摄影师基本功究竟是个啥？

张小北，编剧、导演 查看知乎原文

我不是电影摄影师，但和不少电影摄影合作过。下面都是从个人角度出发讨论的一些看法。

首先，当我们在说「电影摄影师」时，一般来说我们讨论的都是「摄影指导」（DP，Director of Photography）。

DP 这个岗位除了是摄影大部门的负责人，还是最重要的电影主创岗位之一，以及电影剧组的技术总负责人，基本上所有剧组技术部门都要向 DP 汇报工作，而 DP 也对很多重要技术环节和流程有最终决定权。

那么，当我们在讨论摄影师的「基本功」时，其实我们是在讨论一个最根本的核心问题：

电影摄影是干什么的？

通俗地说，在剧组拍摄工作中，「导演负责连戏，摄影负责连光」。

电影摄影要能在最终完成片里，用各种影像手段负责流畅准确地完成画面塑造和氛围传达。具体手段有很多，包括但不限于景别机位构图运动色彩照明等等，本提问下有不少优秀回答，可供大家参考。

最基本的要求，就是电影摄影要善于用光。这就是电影摄影的基本功。

DP 要能在现场巨大的周期压力和限定的工作条件下，落实电影美学，并建立个人风格；然后在这个基础上，DP 再调用所有技术手段，完成电影美学的视觉化。

有一个冷笑话，什么是好摄影的基础？善于用光！

（只有把预算都用光，才能让摄影发挥光影特长。）

这就是为什么电影灯光这么重要的一个部门，一直都是隶属于摄影部门的。如果你作为导演或制片人，要让 DP 去打硬仗，那就得在力所能及的前提下给他配置最好的战士和武器。当然，真正的武士，也会善于运用手头能搞到的一切武器去赢得战斗。

所以，基本功扎实的摄影指导，能在限定周期和资源的前提下，顺利完成电影美学的视觉化，并且在有可能的前提下，还能做出可识别的风格化。

（这种可识别不是为了差异化而差异化，而是基于 DP 对于电影美学和故事的独特理解，并且经过精确设计和落实后呈现的结果。）

关于现场压力……大家可能注意到，我一直在强调「限定周期和资源的前提下」，因为有时不合理周期导致的巨大现场进度压力，有两个直接受害者，一个是导演，一个就是摄影指导了。甚至在某些（关系比较复杂一言难尽的）剧组里，DP 是现场进度压力的直接承受者。

（为什么很多摄影都有胃病？除了吃饭不规律外，就是长期承受精神压力导致的。）

如何面对现场进度压力，也是中国摄影指导所必须练就的基本功之一。（我没和外国 DP 合作过，但我猜想 DP 面临的压力应该差不多，但因为有工会限制，可能会比中国同行稍好一点。）

如何能「多快好省」地完成每天的工作进度，是对 DP 基本功的一个巨大考验。

最后多聊一句。合格乃至优秀的摄影指导 DP 一般都会经过「掌机 / 摄影师」Cameraman/Camera Operator 这个阶段。掌机是在拍摄现场最终落实 DP 所有技术要求的重要岗位，理论上 DP 都是能够胜任掌机这个岗位的。

而且我知道很多 DP 在拍摄一些手持镜头时，为了确保画面节奏和镜头与演员的关系，都会亲自掌机。因为很多微妙的韵律和氛围是无法用语言传递的，犹如透过镜头和演员在谈恋爱，这种事儿必须也只能自己亲自上。

（我之前看过一个八卦，美国摄影师协会为了保证会员的工作机会，是不允许 DP 亲自掌机的。有一些美国摄影指导为了获得工作机会，就想出一个办法来绕开协会限制——甲 DP 这次雇佣乙 DP 做掌机，而乙 DP 下次就雇佣甲 DP 来担任掌机。）

最后的最后，再补充一句，「摄影指导」（DP，Director of Photography）以及「掌机 / 摄影师」（Cameraman/Camera Operator ）和性别无关，有很多优秀的女性摄影师在行业内发光发热。所以我在上文里没有使用「他 / 她」这种说法。

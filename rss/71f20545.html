沽酒家×Quote/0: 将诗句摘到摘录


我运营了一个古诗词的公众号，每天都会选择一首诗词并摘取其中的诗句（摘句）发布到公众号上。当我看到 Quote/0 时，我就在想可以把摘句摘到摘录里呀。

于是购买了 Quote/0，使用了 RSS 和文本 API 功能实现了摘录中展示诗句的功能。

场景一：每日摘句

我基于每天推荐的摘句数据开发了沽酒家 RSS。

RSS链接： https://v1.gujiujia.com/v1/rss

场景二：自定义摘句

我在沽酒家小程序中新增了将摘句发送到Quote/0设备上，这样就可以将自己喜欢的诗句摘到摘录里。操作流程如下：

获取 API 秘钥和获取设备序列号。

在沽酒家小程序中添加设备：我的 - 设置 - 设备 - 添加设备。

发送：诗词详情页 - 摘句 - 操作 - 发送到设备。

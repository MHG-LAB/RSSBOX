用户痛批 Win11 是“纯粹的噩梦”，微软高管回应称将赢回信任

IT之家 6 月 15 日消息，随着 Windows 11 的不断普及，其强制升级的日期也逐渐临近 —— 从 2025 年 10 月 14 日起，所有用户都将无法避免地转向这一操作系统。然而，Windows 11 并非尽善尽美，部分用户对其表达了强烈的不满，甚至有用户因无法忍受其诸多问题而选择切换到其他操作系统。

在社交媒体平台 X 上，一位用户分享了自己对 Windows 11 的极度不满，并称其为“纯粹的噩梦（pure horror）”。该用户表示，从 Windows 8 开始，自己对微软操作系统的失望就与日俱增，Windows 10 稍有改善，但 Windows 11 却让他彻底失望，最终促使他一年多前转投 Fedora 操作系统，并且表示再也不会回头。他列举了 Windows 11 的诸多问题，包括隐私问题、系统臃肿、性能不佳、漏洞频出以及用户界面和用户体验的不足。他举例说，Windows 的开始菜单搜索功能曾经快速且灵敏，但在 Windows 11 中却大不如前。

对此，微软 Windows Insider 计划负责人布兰登・勒布朗（Brandon LeBlanc）回应称，他对该用户因 Windows 11 而离开表示遗憾，并希望未来能够赢回这位用户。他还提到，Fedora 是他最喜欢的 Linux 发行版之一，并表示微软会努力改进 Windows 11，以期让用户重新回归。

尽管 Windows 11 存在诸多问题，但微软在处理用户批评时的态度却值得肯定。从近期的更新来看，微软似乎在某些方面对用户反馈持开放态度。例如，Windows 11 的控制面板功能正在逐步迁移到设置页面，包括字体页面等。此外，用户现在可以通过设置页面管理时间同步的时间服务器，而这一功能此前仅在控制面板中提供。

IT之家注意到，微软还在开始菜单中引入了新的设计，并增加了关闭推荐信息流的选项。这些改进都是基于 Windows Insider 计划参与者和热情的微软粉丝的反馈而进行的。新的开始菜单采用了分类布局，将应用程序分组显示，虽然看起来有点像动态磁贴，但图标仍然是静态的。

微软表示，他们在设计新的开始菜单时充分听取了用户的反馈。通过反馈中心、咖啡交流会和远程访谈等方式，微软收集了大量真实 Windows 粉丝的意见。公司还通过 Figma 和纸质草图创建布局，并与超过 300 名粉丝测试原型，以确保用户体验的流畅性，即使是初学者也能迅速熟悉 Windows 11。

尽管如此，Windows 11 仍然存在一些让用户难以忍受的问题。除了严格的硬件要求和性能问题外，许多用户还对 Windows 11 对 Edge 浏览器和必应搜索引擎的过度推广感到不满。不过，随着微软管理层的调整和组织架构的变化，这种过度推广的现象似乎有所缓解。

对于 Windows 11 的未来，微软仍需继续努力改进，以满足用户的期望。而对于那些已经因不满而离开的用户，微软能否通过持续的改进重新赢得他们的信任，仍是一个未知数。

怎么看待《塞尔达传说 旷野之息》续作跳票将在2023年春季发售？

平昭·南宫司卿的回答

没有确切发售时间的延期现在也不是什么新鲜事儿，何况塞尔达系列本来也是跳票大户，在系列的节点里比较重要的三个游戏就都因为不同原因跳了票：

《时光之笛》原定是 1996 / 1997 年有一套配合 N64 发售的计划（包括马 64 在内），宫本茂决定重做后推迟到了 1998 年；

《黄昏公主》也是原定于 2005 年底发售，推迟一年为了 Wii 的上市首发护航；

还有一个就是《旷野之息》了，原定 2015 / 2016 发售，一路在来回打磨和 NX 计划的双重作用下终于在 Switch 首发。

游戏这种载体，开发是永远没有尽头的，绝不存在能 100% 上线的游戏，一个佳作背后有无数的想法与无数的废案。延期不延期和话语权也有很大关系，就只看还在火热的《艾尔登法环》各种删减的内容都知道这个游戏还有多大的打磨空间，想都不用想也知道制作方来自于发行商死线的压力有多大。倒并不是抨击万代或者哪家发行商，游戏首先是一个商品，商业考量的优先级不用多说。

塞尔达系列在《旷野之息》之前，市场反馈肯定是不能算任系头牌的，但塞尔达系列对于行业与核心玩家的影响力可以说是数一数二的，所以在关键点上，延期往往是必要的考虑因素——宫本茂的话语权也是一大方面，说实话在 2D 转 3D 的整个家用游戏行业最关键的时期，《时光之笛》能抛开 N64 的发售计划跳票，没有顶级的话语权，和足够的魄力，是做不到的。

《旷野之息》则不尽然，后半段 WiiU 属于战略放弃投入到 NX 虽然不是赌博行为（任天堂的家底还是够玩脱两三世代的），但某种意义上也是为这作有了更大的发挥和调整空间；至于目前一路高歌猛进的 Switch，《旷野之息》的续篇并不承担什么对装机量的责任。

因此不管是悲观来看开发过程不顺利需要改变，还是乐观来看有更多的时间打磨基于无比优秀的 1 代框架的新内容，都能够让人更期待一下续篇的成品，尽管等待的时间总是焦急的。

青沼接受 kotaku 采访的时候也说过，他们想做一些「新」的东西，但并不知道要花多久以及什么样才是「新」；而宫本茂那句行业内至少是 Top5 级别的名言更有说服力：

A delayed game is eventually good, but a rushed game is forever bad.

（虽然绝大部分开发者既没有任天堂的环境，也没有宫本茂的话语权……）


话说回来，从塞尔达的 3 个重要作品跳票也看得出来，一般任天堂重要 IP 的跳票就两个因素，保开发或者为了新机护航，可能也会有人觉得这是在给加强型 Switch 吹风，但从 18 年到现在，我们眼睁睁看着 Lite 和 OLED 出世，传闻一直嚷嚷的加强型 Pro 还未有任何痕迹，这种猜想恐怕也难成真……


至于 goty 的玩笑，不会真有人当真吧。

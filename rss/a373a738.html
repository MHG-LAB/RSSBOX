现阶段国产手机都推出了折叠屏，苹果为什么迟迟没有动静？

李昂的回答

随着vivo X Fold发布，国内排名前五的厂商也就剩下苹果没有发布折叠屏手机了。可以肯定的告诉大家苹果的折叠屏也在设计之中，那苹果折叠屏又会有怎么样的特点？正好我手上也有最新的vivo X Fold，我就借着vivo X Fold的使用感受给大家讲一下折叠屏的设计思路，这些思路也会在未来的折叠屏中见到。

外屏

折叠屏最大的意义我觉得首先是基于现有手机的扩展，展开之前的小屏应该提供一个更接近传统手机的体验，这也是高端商务旗舰的必备因素。像vivo传统手机的屏幕比例基本上都是20比9，而vivo X Fold不展开的时候是一块22比9的屏幕，这块屏幕能够提供更接近20比9手机的用户体验，用户习惯它还是比较容易的。

再加上这几年各家厂商都是在不断拉宽屏幕，从非全面屏时代的16比9到如今的20比9甚至20.5比9，22比9的屏幕颇有一种未来已来的感觉。而 vivo X Fold采用的内外120Hz E5双主屏也能为日常使用提供旗舰级别的亮度表现，再加上获得19项Display Mate A+认证，在屏幕硬件方面堆料十足的vivo X Fold无疑是相当优秀的存在。

内屏

折叠屏内屏最大的场景我觉得还是提供一个更接近平板端的体验，不管是操作逻辑还是追剧生产力都更接近平板。刚刚发布vivo X Fold它的内屏采用的是一块16比11的8英寸大屏，这个比例和16比10的vivo Pad基本一致，这个尺寸也非常适合追剧。再加上vivo对折叠屏做出的系统适配可以非常有效的利用这块大屏，二级菜单等方面的适配也能够提供一个非常优秀的操作体验。

事实上vivo X Fold的8英寸屏幕非常接近苹果的ipad Mini6的8.3英寸了，也确实能够非常接近平板的体验。vivo X Fold在观看视频的时候除了可以获得一个更接近平板的大屏观感之外，还可以利用它的航天级浮翼式铰链支持多角度悬停的特性将下半部分变成操控台，进而去调整进度和声音亮度等设置，这个功能还是非常新颖实用的。

指纹识别方面vivo X Fold的堆料也相当强悍，它拥有目前折叠屏中独一份的3D超声双屏幕指纹，不管是内屏解锁还是外屏解锁都相当优秀。当然这项技术实现起来也相当不容易，折叠破本身需要金属盖板来保护屏幕，所以传统的光学指纹和超声波根本不可能使用，vivo利用超声波指纹和盖板共振的原理才实现了这个功能，这个思路相当巧妙。

说完了屏幕部分再说说充电部分，vivo X Fold随机附赠一颗80w的氮化镓充电器，这款充电器最高支持65W PD快充，给vivo X Fold充电最高支持66w，实测下来40分钟就可以冲入。因为采用了可折叠式的设计平时不用的时候完全可以将插头折叠起来，减少空间占用。无线方面vivo X Fold还支持50w无线闪充，也是折叠屏中少见的支持无线充的机型。

拍照部分vivo X Fold堆料也相当优秀，蔡司专业影像加持下的拍照体验相当优秀，再加上蔡司T*镀膜保证了拍照上限相当高。色彩还原相当准确。这一代vivo X Fold的拍照的色彩饱和度相当不错，基本上随手拍都能拍出来非常优秀讨喜都照片。主，广，人，长四个焦段齐全也意味着不管什么场景都可以轻松驾驭，也是2022年少有的焦段齐全的拍照旗舰。




总结

vivo X Fold可以说是目前为止我个人认为非常不错的产品，它的120hz外屏能够提供接近手机的操作体验，而120hz的内屏又可以提供接近平板的操作体验，再加上超声波指纹识别和强悍的相机堆料，后发先至的vivo交出了一个相当不错的答案。

后发先至，其实也是苹果的强项所在。

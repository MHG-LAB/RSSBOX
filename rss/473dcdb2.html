    # 2024.10.17

    今日要闻

    Unity 6 正式发布：大幅提升稳定性和性能、渲染速度更快

    Unity 6 已正式发布，与上一个主要版本相比，Unity 6 大幅提升了稳定性和性能，并且为所有 Unity 客户提供更强大的支持。这是一个 LTS 版本，官方承诺在下一个主要版本发布后至少再提供两年的支持服务。

    Unity 6 有许多令人兴奋的新工具和功能，让开发者体验更快的渲染速度、更强大的光照控制、端到端的多人游戏工作流程、智能化的动态 AI 功能，以及对移动 Web Runtime 的更多支持。

    • 端到端多人游戏工作流程将加速联网游戏的开发；
    • 各种工具可以助力开发者开发移动 Web 项目；
    • 新的图形功能可以将工作负载从 CPU 转移到 GPU，例如 GPU Resident Drawer 和 Render Graph for Universal Render Pipeline (URP) 功能，在内部测试和客户测试中将 CPU 性能提高了多达 4 倍。

    开源不足一月，WinAmp 仓库已 404

    在宣布开源不到一个月后，WinAmp 所有者突然将整个 GitHub 库进行了删除。而仓库中一部分可能来自于组织外部其他贡献者的源码，也随之消失不见。

    目前，https://github.com/WinampDesktop/winamp 页面已经 404。

    Winamp 是 Nullsoft 于 1997 年开发推出的一款多媒体播放器，以其灵活性和对音频格式的广泛兼容性而闻名，至今仍拥有数百万用户。项目团队在今年 9 月份兑现承诺，正式在 GitHub 上发布了其 Windows 端的源代码。但采用的 Winamp Collaborative License（WCL）Version 1.0 协议，对用户的分叉以及修改后的分发出了限制。

    Azure OpenAI 即将关停对中国个人用户的服务

    微软最近发布公告，通知部分 Azure OpenAI 用户服务即将终止。

    根据新的规定，只有企业客户可以继续订阅和使用 Azure OpenAI 服务，这一变更将在 2024 年 10 月 21 日正式生效。这意味着 Azure OpenAI 服务不再提供给个人用户

    ChatGPT 登陆 Windows

    OpenAI 宣布已开始预览其 AI 聊天机器人平台 ChatGPT 的专用 Windows 应用程序。

    下载地址：https://openai.com/chatgpt/download/

    几天过去，服务器转码应该稳定了

    自人类发明视频伊始，我们就一直走在追求高画质与低成本的权衡之路上，早在阿B还在占用优酷服务器发视频的时候，就有UP主捣鼓“后黑学”来提高整体视频码率了。AVC和HEVC作为新兴技术出现的时候，也经历了很长一段的阵痛期，也许AV1因为巨头都在推也会成为主流标准，但谁都说不准。这条博文仅作技术分析之用，因其特殊性不会做成视频，也不希望大家过分攻击平台，特此套盾。

    对了，谷歌毕竟是AV1的主导者和推行者，所以拿油管默认AV1/VP9来跟国内平台对比并不合适，特此二次套盾。

    对对了，除了锐化，某些平台甚至在尝试使用AI算法来后期强化画质，把画面弄得跟蹩脚大模型生成的一样，哈哈哈哈哈特此三次套盾。

    *注：VMAF（Video Multimethod Assessment Fusion）是由Netflix开发的一种视频质量评估工具，不同于传统数学方法，VMAF使用更注重肉眼效果的机器学习模型来评估画质。

    LeCun称已基本放弃语言模型

    当奥特曼、马斯克、Anthropic CEO都纷纷将AGI实现锚定在2026年前后，LeCun无疑是直接浇了冷水：完全是胡说八道。

    最新的公开演讲中，他表示，系统要推理、规划和理解物理世界，至少还需要几年甚至十年的时间，这时候人工智能才能达到人类水平。

    他还透露，现在Meta基本已经放弃纯语言模型，因为，仅通过文本训练，永远不会达到接近人类水平的智能。

    除此之外，他还谈到当前大模型的局限性、以目标驱动的AI架构，包括利用世界模型进行推理和规划，最后强调了开源AI的重要性。对于未来发展，他提到，最终机器将超越人类智能，但它们将受到控制，因为它们是目标驱动的。

    Kimi说要重新定义搜索，但还为时过早

    也许是考虑到二十年前的技术和现在的大数据、AI、移动互联网不可同日而语，于是，在当前的搜索领域，人们提出新的搜索理念和产品，甚至会觉得，新兴技术能够颠覆传统搜索数十年的积累。

    但现在的AI搜索，更多是聚焦在狭义范围，认为只要改变用户的交互方式，提高搜索结果的相关性，就能实现“重新定义”。但这其实应该被称为“答案引擎”，而不是真正意义上的“搜索引擎”。

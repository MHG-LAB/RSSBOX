Level Read：让英语学习从「读得懂」开始，每天都进步一点点


Matrix 首页推荐

Matrix 是少数派的写作社区，我们主张分享真实的产品体验，有实用价值的经验与思考。我们会不定期挑选 Matrix 最优质的文章，展示来自用户的最真实的体验和观点。

文章代表作者个人观点，少数派仅对标题和排版略作修改。

很多人学英语的过程，像是在不断完成任务：背单词、刷题、啃外刊、挑战原著。但我一直觉得，语言不应该只是用来考试或者折磨自己的。语言首先是一种工具，它应该帮我们读懂一本书、听懂一个故事，或者了解这个世界正在发生什么。

这也是我做 Level Read 的初衷。我希望它能让你每天轻松读一点英文内容，而不是为英语学习这件事感到焦虑。你不需要一开始就读《经济学人》，也不需要每遇到一个生词就停下来查半天。你只需要从适合自己难度的内容开始，每天读一篇、听一遍、理解一点、积累一点。

Level Read 有网站版本，也有 iOS 和 Android App。你可以在电脑上阅读，也可以在手机上把它当成一个每天打开几分钟的英语阅读工具。

让「学英语」回归阅读

很多英语学习材料都太难了。

外刊当然很好，但它们本来不是为学习者写的。很多时候我们打开一篇英文文章，会发现最花时间的其实是不停查词、翻译长句、猜测上下文。读完一段已经很累了，更不用说每天坚持。

英文原著更是如此。它们很有吸引力，但对大多数学习者来说，门槛并不低，很难找到一个刚好适合自己的入口。

所以 Level Read 想解决的第一个问题是：能不能让一个普通英语学习者，每天都读到一篇自己能读懂的英文内容？

我不希望用户一开始就被高难度材料劝退。更理想的状态是，读完一篇文章后你真的知道它在讲什么，你能从内容里获得信息，你能感受到自己是在使用英语，而不是被英语测试。

这种感觉很重要。语言学习的持续动力，很多时候其实来自「我今天真的读懂了一点什么」。

一篇新闻，三个难度

Level Read 最核心的设计，是把同一篇新闻改写成三个不同难度。

  • Level 1 更简单，适合刚开始建立英文阅读习惯的学习者
  • Level 2 会更自然一些，适合有一定基础的人
  • Level 3 则更接近真实英文表达，适合想继续进阶的用户

你不需要一上来就挑战最难的内容。你可以从自己能读懂的 Level 开始，先把文章读完。如果你愿意，也可以再切到更高一级，看看同一个意思在更自然的英文里是怎么表达的。

这也是我很喜欢分级阅读的原因。它不会把内容变得幼稚，而是给学习者一个可以进入内容的台阶。你不必在「太简单」和「太难」之间二选一，而是可以从适合自己的地方开始。比如同一篇关于科技、文化、动物、环境或者国际事件的新闻，不同水平的用户都可以阅读。基础弱一点的用户可以先读懂大意，基础好一点的用户可以关注更自然的表达。大家读的是同一个世界，只是入口不同。

就拿今天最新的这篇新闻 America's Schools Face a Backlash on Digital Devices（美国学校面临数字设备的反弹）举例：

当然了，Level Read 的分级阅读并不是随便找一篇文章，然后让 AI 简单「写得简单一点」。目前 Level Read 的内容生产，首先要经过人工筛选来源，我们会从 BBC 等国外主流英文媒体中选择适合学习者阅读的新闻内容，相比特别复杂、强时效、专业术语密集，或者需要大量背景知识才能理解的报道，我们更倾向于选择信息清楚、主题相对普遍、适合普通学习者理解的内容。

在选定素材后，Level Read 会使用大语言模型辅助完成分级改写。Level Read 已经上线一年半左右，目前有 3 万多名用户使用。在这个过程中，我们也一直在根据模型能力的变化、业内模型的实际效果，以及用户使用中的反馈，持续调整内容生成流程。

目前，Level Read 主要使用 Claude Sonnet 4 系列模型辅助完成分级改写和学习内容生成。改写时，我们不会只要求模型「降低难度」，而是会围绕不同 Level，参考 CEFR 标准，对词汇难度、句子长度、语法复杂度、信息密度和文章长度做控制。

在提示词设计上，我们最重视三件事：

  • 第一，不改变新闻事实，不添加原文没有的信息。
  • 第二，不为了降低难度而过度简化，导致内容失真。
  • 第三，不把文章改写成机械的教材腔，而是尽量保持自然、清楚的英文表达。

从实际使用效果看，Claude Sonnet 在这类任务上比较适合 Level Read：它对长文本结构的保持比较稳定，改写后的英文也相对自然，不容易把文章改成过于生硬的教材句。同时，它在不同难度之间的区分也比较清楚，能更好地把同一篇内容改写成适合不同水平学习者阅读的版本。

当然，AI 并不意味着内容可以完全无人参与。对 Level Read 来说，AI 更像是辅助生产工具，真正重要的仍然是前期选题、难度控制、内容检查，以及后续根据用户反馈不断修正。

目前 Level Read 暂不支持 BYOK，也不支持用户自定义模型和改写提示词。原因是现阶段我们更希望它是一个普通学习者打开就能直接阅读的分级阅读产品，而不是一个需要先配置模型、Prompt 和参数的 AI 工具。未来如果开放导入任意文章、自定义学习材料等能力，BYOK 和自定义提示词会是值得认真考虑的方向。

帮你读懂一篇文章

在 Level Read 里，阅读是一件尽量连续的事情。

打开一篇文章后，你可以选择适合自己的难度开始阅读。遇到不认识的词，可以直接点词查看解释；如果想确认整篇文章的意思，也可以查看全文翻译。每篇文章也配有音频，你可以边读边听，或者在通勤、散步的时候只听音频。读完之后，还有一个简单的阅读理解 quiz，帮你确认自己是不是真的读懂了这篇文章。

这些功能都围绕一个目标：帮助你完整读完一篇英文内容。

我不希望 Level Read 变成一个到处弹出功能、打断阅读节奏的工具。查词、翻译、音频、quiz 都应该服务于阅读本身。

单词本与阅读情境

Level Read 也有单词本，所有生词都来自你正在阅读的新闻。

当你在文章里遇到一个不认识、或者觉得值得记住的词，可以把它加入单词本。这个动作不是为了马上开始背诵，而是为这个词留下一个来源：你是在什么新闻里遇到它的，它当时出现在什么句子里，它为什么值得被记住。

这和我自己对单词学习的理解有关。孤立地背单词当然有用，但对很多人来说，它很容易变成负担。一个单词、一条中文释义、一个例句，然后反复记忆。时间久了，单词好像记住了，但一放回真实文章里，又变得陌生。

我更希望单词是从阅读里长出来的。当你把一个词加入 Level Read 的单词本后，如果它以后再次出现在新的新闻里，App 会自动把它高亮出来。你会很自然地意识到：原来这个词又出现了。

一个词，多段文本

Level Read 也提供单词 review 功能，在 review 页面里，每个单词都会显示释义、发音和音标。除此之外，你还可以看到它最初来自哪一篇文章，以及当时出现在哪个句子里。

更进一步，Level Read 还会整理出所有包含这个单词的新闻和句子。也就是说，你可以从一个单词出发，看到它在不同新闻场景下是如何被使用的。

比如一个词第一次出现在一篇科技新闻里，后来又出现在经济新闻、环保新闻，或者人物故事里。你会慢慢发现，同一个词在不同语境下有细微不同的用法。

这个过程比单纯记住一个中文释义更接近真实语言学习。

每天一点点，慢慢变成习惯

对大多数人来说，学英语最难的地方在于长期持续。

所以 Level Read 的节奏设计得比较轻。每天读一篇文章，听一遍音频，顺手收集几个词，再做一个简单 quiz。它不需要你一次投入很长时间，也不希望你因为某天没学就产生很强的负罪感。

我更希望它成为一个可以放进日常生活里的小习惯：早上通勤时读一篇，午休时听一遍，晚上睡前复习几个词。每天 5 到 10 分钟，持续半年，可能比偶尔热血三天更有意义。

Level Read 里也有学习记录和进度追踪，可以帮你看见那些很小但真实的积累：读过多少文章、听过多少内容、积累了哪些词、连续坚持了多久……语言学习很少有突然开窍的瞬间，更多的时候，它是一次次读懂、一点点熟悉、慢慢不再害怕英文内容。

网站、iOS 和 Android 都可以使用

英语学习不应该被某一个设备限制。很多人白天在电脑前，晚上在手机上；有时候想认真读，有时候只是想听一听。所以 Level Read 目前有网站版本，也有 iOS 和 Android 版本。如果你习惯在电脑上阅读，可以直接打开网站；如果你希望把它变成每天随手打开的学习习惯，也可以使用手机 App。

 

除了在网站和 App 里阅读，你还可以把每篇内容导出成 PDF。这个功能最初是为了满足一些更偏「学习资料」的使用场景。比如你想把某一篇新闻保存下来，之后离线阅读；或者你习惯在 iPad 上批注、划线；又或者你是老师，希望把一篇适合学生水平的英文新闻打印出来，作为课堂阅读材料。

导出的 PDF 会保留文章内容和对应的学习信息，让一篇新闻不只是 App 里读完就过去的内容，也可以变成一份可以收藏、打印、批注和复习的英文阅读材料。

我自己很喜欢这个功能的一点是，它让 Level Read 不只局限在手机里的碎片阅读。你可以在手机上发现一篇文章，也可以把它导出，在电脑、平板、打印纸上慢慢读。对一些更认真学习的人来说，PDF 是一种很自然的沉淀方式。

Level Read 适合谁？

如果你已经习惯每天阅读英文原文，Level Read 可能不是最硬核的工具。但如果你是下面这些人，它可能会很适合你：

  • 你想重新开始学英语，但不想再从单词书第一页开始。
  • 你觉得外刊、原著太难，希望先从更友好的英文内容建立阅读习惯。
  • 你有一定基础，但希望每天稳定接触英文，保持阅读和听力语感。
  • 你不想只为了考试学英语，而是希望真的用英语了解世界。

我做 Level Read 的初衷，是希望提供一个更轻、更容易开始的选择。英语不一定要从痛苦开始，它也可以从一篇你能读懂的新闻开始。

最后

最近经常有人在讨论：有了 AI 翻译，我们还需要学英语吗？我觉得仍然需要。

翻译当然很方便，但能直接读懂另一种语言里的内容，仍然是一件很酷的事情。你不只是得到一个被转换后的结果，而是在更接近原文的地方理解信息、感受表达、建立连接。

Level Read 想做的，就是帮更多人降低这个开始的门槛。你不需要一开始就读最难的内容，不需要把英语变成一件很痛苦的任务。每天读一篇适合自己难度的英文新闻，听一听，查几个词，慢慢积累。

也许半年后你回头看，会发现自己已经读过很多文章，也认识了很多以前总是跳过的词。更重要的是你会发现：英语不再只是一个要被学习的科目，它真的可以成为你理解世界的一种方式。

Level Read 上线了一年多的时间里，收货了 3 万多位有着共同理念的朋友，大家放下了十几年一直攥紧「势必学好英语」的拳头，不再打量和焦虑英语这扇窗户本身，而是走近它、打开它，开始真真切切地去欣赏和感受窗外的世界。

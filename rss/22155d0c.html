怎么能看出一个人开车水平高低？

小林，想把汽车变成爱好 查看知乎原文

巴菲特好像说过以下类似的话，不一定准确：

如果你在牌桌上打了几轮后还没发现谁的水平最差，那么很有可能你就是那一个水平最差的。

放在开车上：如果你看不出别人开得好不好的话，那很有可能你自己的开车水平也不会很高。

而很多自己开车好的人，开得快的人却不一定能够看出别人开得好不好，更别说能够说出好在哪里，不好在哪里。

说实话，能完全看出别人开车水平高低的都是高手中的高手。

反正我肯定做不到。

如果非得看的话，以我现在渣渣的认知水平，也许可能可以从以下几点来看：

注 1：个人看法，不一定对，很有可能错

注 2：没能力系统的写，想到哪写到哪

  • 1.安全：对所有安全因素的识别与恰当措施的采取
  • 2.丝滑：对驾驶及车辆的熟悉→熟练→熟记与脊椎
  • 3.游刃有余：基于预测及经验的提前想三步的驾驶规划
  • 4.速度追求：实时基于当前道路 / 天气 / 车辆 / 自己的情况，最大程度地把动力传输给地面

以上按序号递进，越往上要求越高，一个一个说。

1.安全第一，不是说说而已。

我可能在别的回答或者文章里面都重复过很多遍。

安全第一，安全是我们所有 0 前面的那个 1。没有安全，开得再快再好也没用。

而如何才能开得安全，得先识别出周围的危险因素，然后基于这个危险因素采取适当的操作。

危险因素一直都在那里，而为什么有些人看不到。

第一层是没有去主动识别风险的意识（主观能动性问题）

第二层是不知道如何去识别风险（客观的能力与经验问题）

而，主观能动性可能是最难搞的一部分，要么愿意看、要么不愿意看。

不愿意看的，怎么洗脑，怎么说清楚利益弊害都没用。本性难移也不是说说而已。

如果能过这一关的话，怎么去识别风险我们迟早都能掌握的。

那我们驾驶中的风险有哪些：车辆自身、驾驶者自身、周围车、周围的人、周围的环境天气道路情况。

Case 不能穷举，每个方面说几个例子作为索引：

·车辆本身：上车前会不会大概看下车辆有没有异常（漏气了没，漏油了没）；驾驶的过程中能不能及时发现车辆的异常（方向盘是不是发抖、车子是不是一直跑偏）

·人本身：是不是已经无法集中注意力了（脑子在想着下班前被老板痛骂的事）；是不是体力已经明显下降了（已经长途奔袭 2 小时以上）

·周围的车：是不是夹在两辆大车之间（那么就要尽快的离开这个死亡谷）；是不是右前有两台车一台慢车一台快车（那么快车很有可能会突然变道到你这边来，最好还是等他们稳定了我们再决定要不要超过去）

·周围的人：有没有小孩在路上追逐玩耍，手上脚下还有个球状物体（他们可能不会管路上有没有车就冲到马路上了，我们还是降低车速的好）；中央绿化带上是不是站着人（他们很有可能会突然走下来，我们还是降低车速靠中间车道走的好）

·周围的天气环境道路情况：跑山路看看山体是不荒芜的，路边是不是堆着掉下来的石块（那么我们还是在安全的车速下尽快通过这个危险去）、前方是不是有水潭（那么我们在不知道水有多深的情况下最好躲着走，或者先慢下来看看别的熟悉这条道的老司机怎么开得，实在不行再慢慢的自己上）

好了，总结来说，我们怎么去看一个人开得安不安全：

·看他知不知道自己的车的行驶状态是不是正常的，知不知道自己现在的精神和身体状态适不适合驾驶。

·首先要看他开车的时候有没有一直眼观六路耳听八方，不是看美女看美景的那种看，是一直观察周围有没有风险的看。

·然后看他发现风险后，能不能采取恰当的措施。是发现了风险以后慌慌张张突然变得不会开车了，还是沉稳地刹车 / 变道 / 降速 / 提速 / 鸣笛 / 闪灯 / 甚至打手势。

·最后，再看他的操作是什么风格，是铤而走险的流派还是宁等两分钟不抢一秒的流派。





2.有它，故手熟

这里玩个文字梗：

我们小学都学过卖油翁的故事，也记得里面老翁说过的一句话：「无它，但手熟尔」

不过真的想做到手熟，光靠无它的重复可不行，得刻意练习才可以。

所以我更愿意说成：「有它，故手熟」

这个它：就是我们平时对于自己操作及操作所会带来的车辆响应的刻意观察及迭代。

·方向盘抡快了，车子会怎样:车子是不是会一下子打摆子，让你副驾的领导可以在那么一瞬间可以与你亲近那么一点点。

·如果再慢一点，又会怎样:是不是刚刚好了？会不会就拐不过去这个弯了，需要临时修正或反复修正?

·油门刹车跺得凶了会怎样：会不会把娃正在啃的冰淇淋变成他变成大花脸的涂料，会不会把车子搞得一串一串让老人每次坐车都晕车再也不愿意坐车和你出门玩了。

·如果跺得再慢一点会怎样：是不是就刚刚好了，还是就与前面的小姐姐的车亲密接触了，还是就过不去这个弯了。

·离合：诶这个估计大多数人用不上，也说一说吧。能不能每次都刚好松到半联动的点，大概半联动多久以后就可以松开了，现在离合是不是打滑的特别厉害。

这些都需要我们不断地刻意观察、刻意修正。

直到能够真正的随心所欲的驾驭你的车，然后再多换几台车开开，可以做到无缝切换。

直到这些都变成你的肌肉记忆。

所以我们看一人开车开得熟不熟练：

·是不是所有的操作都恰到好处，不多也不少，不快也不慢。
·是不是所有的操作都如德芙般丝滑，动作姿势都到位的同时，不粗也不糙。
·是不是换了一台车也能够很快的适应，迅速做到上面两步。

3.提前想三步，才能游刃有余

先举个例子：

有些较高阶驾驶训练，会被要求的是在不允许踩刹车（画黑板，这里是重点）的情况下。

这各种各样的车各种水平的驾驶者一同行驶在相同的场地里，会被要求完成类似：

从低速 60kmph 到高速 180kmph 的车速反复变换；变道；汇入车流，汇出车流等等所有考试所需的操作。

关键是你只能用油门和方向去控制这些，而且还得是如德芙般丝滑的去完成这些。

这就不仅仅要求你充分做好预判了，有可能提前想三步还远远不够。

还需要结合自己的经验，你之后三步所需要的做的动作所要去的地方，在那个时空可能会出现什么车，这些车会有什么举动倾向做出预判。

我们看一个人开车是不是游刃有余，就看他除了非常非常偶然的 Corner Case 外。

无论在什么交通环境下，无论在什么车速下，不管晴天还是下雨。

只要不是山无棱，天地合。

他都永远能够从容不迫的驾驶着车带你到达目的地。

而，这些非常非常偶然的 corner case，如果真的做到了提前想三步 + 不断刻意积累的脑中你关于不同车不同环境的大数据的话，也需可以尽可能的避免。

不伤害他人，同时不被别人伤害。

如何做到不被别人伤害，才是最难的。

哪怕别人是马路杀手，哪怕是对方全责，但受伤的还是我们自己，保护自己人人有责。

所以再看一眼的话：就是他开车时愿不愿意为了保护自己，而对一些周围不合理的驾驶行为做出合理的保护自我的驾驶方式。

OK，关于日常驾驶其实 1~3 说得就够了。

说是说了，但如何知道做到，尤其在没有教练在旁边的时候如何做到自知自觉。

如果知道做不到，知道再多也没用，永远都无法进步。

关于自己开得好，怎么看出来别人开得好，能不能真的教别人开得好。

一层层递进，一层层变难。

不过只要有心的话,都还没要到拼天赋的地步，基本大部分拿了驾照的人都变成一个开车开得很好的人。

4.关于速度的追求游戏

如果把以上的步骤都做好了，而且还对驾驶很感兴趣的话，可能就会去追逐一些竞技驾驶技巧了。

竞技驾驶就是为了更快。但是为了快，为了持续的快，为了在任何环境任何车辆中的快。

上面 1~3 的判定基准其实都适用。

到了竞技上不过是把这些的精度、稳定度、各种操作的 timing 都打磨到精益求精而已。

只不过这个精益求精需要极大的精力时间金钱 - 最后天赋去堆砌出来。

晚点LatePost：【晚点独家丨蔚来资本领投 Arm CPU 公司…

晚点LatePost：
【晚点独家丨蔚来资本领投 Arm CPU 公司此芯，投后估值约 3 亿美元】中国市场自去年来涌现了十余家 Arm CPU 创业公司。《晚点 LatePost》独家获悉，Arm CPU 公司此芯科技近期完成了约五千万美元的新一轮融资，由蔚来资本和启明创投共同领投，云九资本、基石资本、中科创星、嘉实资本、贝塔斯曼亚洲投资基金、元禾璞华跟投，投后估值约 3 亿美元。这是此芯成立 9 个月来的第四笔融资。
大多数创业公司想从服务器 CPU 切入市场，如鸿均微、启灵芯、遇贤微等。此芯的第一款产品则是电脑 CPU，对标苹果 M1。
此芯创始人孙文剑是 AMD 前客户定制部门中国区负责人。他告诉《晚点 LatePost》，在苹果于 2020 年 11 月发布电脑 CPU M1 之后，他认为 Arm CPU 的创业时机已成熟。
过去，Arm 架构 CPU 主导手机、平板等移动设备市场，英特尔主导的 x86 架构占据利润更高的个人电脑、服务器市场，界限分明。但近年来，Arm 逐步进入 x86 的优势领地。亚马逊 2018 年推出的服务器芯片 Grivton 和苹果后来发布的电脑芯片 M1 都基于 Arm 架构。
这之前，试图把 Arm 用到服务器和电脑上的尝试已有多年，参与者包括高通、AMD、Marvell 等，但成果并不明显。2016 年末，全球手机 CPU 龙头高通开始开发支持 Windows 10 的 Arm 电脑芯片，但其与微软合作的 Surface Pro X Arm 版对部分工作软件支持不佳，未获得良好市场反馈。
孙文剑认为，这些尝试的失败是因为当时 Windows 等主流电脑操作系统对 Arm CPU 支持不足，软件生态不完善。
而随着 2020 年年末，苹果推出 M1 并收获良好市场反馈，孙文剑预测，多年来与英特尔 x86 CPU 深度绑定的微软 Windows 操作系统会加速跟进这一变化，更积极支持 Arm 生态。微软和大部分 Windows 电脑厂商并不会像苹果那样自研芯片，这是第三方芯片供应商的空间。“这是一个没法错失的机会。不创业会后悔一辈子。” 孙文剑说。
苹果推出 M1 9 个月后，此芯于 2021 年 10 月注册成立，创始成员多有共事经验，各自负责芯片设计中的关键环节。除孙文剑外，此芯联合创始人兼 CTO 刘芳、芯片工程负责人沈朝晖、软件工程负责人刘刚和系统工程负责人褚染洲都曾任职于 AMD。
刘芳此前还曾任职于苹果，参与 M1 的开发。她于 2005 年加入 P.A.Semi 从事芯片内核开发，随着 2008 年苹果收购 P.A Semi，刘芳进入苹果担任芯片架构师；2019 年后，刘芳先后任职 AMD 和 Meta。在 P.A Semi 和苹果期间，刘芳团队的负责人是芯片设计领域传奇人物 Jim Keller，他曾领导苹果手机芯片 A4、A5、AMD 的翻身之作 Zen 架构和特斯拉自动驾驶芯片 FSD 的开发。
在去年下半年的筹备阶段，此芯面临两个不明朗的环境因素：一是当时英伟达对 Arm 的收购仍在进行，如收购成功，为众多芯片公司提供 IP 的 Arm 会成为英伟达独占产权，其开放度可能受限；二是 Windows 在 2021 年仍未表露支持 Arm 的动作，孙文剑判断，微软受 M1 刺激继而拥抱 Arm 的迹象还不够明显。
今年以来，事情朝着有利创业公司的方向发展。今年 2 月，由于大部分芯片公司和多国政府反对，英伟达终止收购 Arm。同月，微软宣布加入基于 Arm 的开源组织 Linaro，微软 CVP Kevin Gallo 在发布会上说，将与 Linaro、Arm 和高通一起推动基于 Arm 的 Windows 生态系统。5 月的 Build 开发者大会上，微软进一步宣布已在开发 Visual Studio 2022 和 VS Code 的本地 Arm 64 版本， 这是一个基于 Arm 开发的工具链，可帮助开发者节省 Arm 软件的开发时间和流程。
电脑生产厂商也在更积极布局 Arm CPU。此芯第一个投资方是联想创投，孙文剑告诉《晚点 LatePost》，在接触到此芯前，联想已在多方寻找做 Arm CPU 的团队。
成立不到 1 年的此芯当下最重要的目标是先做出第一款芯片，用 1-2 年完成商业化，再逐步拓展新的应用场景和产品线。此芯的第一款电脑 CPU 目前处于工程设计阶段。
联想投资对此芯商业化有一定助力。联想去年共卖出 8190 万台个人电脑，市占率 23.5%，位居全球第一。一位行业人士评价，这让此芯少了一些找客户的焦虑。#每日新闻#

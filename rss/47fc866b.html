这是一个六年前提出的好问题，尽管很多答主如章佳杰，猪小宝等都贡献了有意义的回答，但其实并未真正意义上解决问题。在认真阅读完大部分回答后，今天就由我来终结此问题。

提炼一下最关键的两个问题，

（1）钢丝挂肉，到底是两头重中间轻好，还是两头轻中间重好？

（2）是否存在一种最优的挂肉方法，使钢丝的的承载力最小？

至此，两大问题均获得了解决。不过，解答中作了一些不影响结论的假设，虽然抓住了问题的主要特征，但实际问题千变万化，为了应对更复杂的情况，下面对这些假设作进一步的分析。

（1）假设一：载荷沿坐标轴分布。实际上，更多的情况是载荷沿绳长分布，此时只需将 替换为 ，不增加问题难度。

（2）假设二：绳子总长度不变。有人争辩说需要考虑绳子的弹性，这也不难，具体可参考我的一个回答（弹性悬链线问题的求解）。

（3）假设三：绳长约束方程的泰勒展开。此处假设见公式（3），作此假设只是为了获得泛函的显式表达式，如舍弃此假设，仅仅多一步隐函数求导，亦不影响最后结论。

最后总结一下，在物理上，显然知道将载荷放在端点上，将会使绳子承载力最小，但这个结果太平凡了，一度不被认为是合理解，尽管最后使用弱解将其抢救了回来，也只能说勉强接受吧。个人的看法是，这个问题或许有更好的提法，感兴趣的同学可以想一想，如果真能想到的话，可以邀请我来回答，哈哈！

Clear["Global`*"];
L0 = 0.5033; L = 0.5;
g = 9.8; M = 100;
q0 = 200; k = -400;
\[Rho][x_] := q0 + k x;
sol[H_] := 
  NDSolve[{ y''[x] == \[Rho][x]*g/H, y[0] == 0, y'[0] == 0}, 
    y, {x, 0, L}] // First;
dH = 10^-6; H = M g;
Do[
  Print[H];
  fp = NIntegrate[Sqrt[1 + y'[x]^2] /. sol[H + dH], {x, 0, L}] - L0;
  f = NIntegrate[Sqrt[1 + y'[x]^2] /. sol[H], {x, 0, L}] - L0;
  df = (fp - f)/dH;
  H = H - f/df, {i, 1, 20}];
ParametricPlot[{{t, y[t] /. sol[H]}, {-t, y[t] /. sol[H]}}, {t, 0, L}]

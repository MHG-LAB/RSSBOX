GitHub Sponsors Launches in Brazil

Leia este artigo em português

GitHub sponsors é lançado no Brasil

Nosso mundo caminha baseado em código aberto. É por isso que há três anos lançamos o Sponsors para que pessoas desenvolvedoras de Software e organizações que usam o open source pudessem receber apoio financeiro para o importante trabalho que fazem todos os dias.. 

Enquanto a comunidade open source continua a crescer – um rápido para os 10 milhões de novos desenvolvedores que se juntaram ao GitHub nos últimos 6 meses! – queremos ter certeza de que cada desenvolvedor tenha oportunidades iguais.

É por isso que estamos felizes em anunciar que o GitHub Sponsors está agora disponível no Brasil – uma expansão empolgante para uma das nossas comunidades desenvolvedoras de crescimento mais rápido. Desenvolvedores já inscritos na lista de espera estão agora no programa e aqueles com contas bancárias no Brasil podem criar perfis para começar a receber financiamento.

A comunidade open source do Brasil

Enquanto o crescimento está acontecendo globalmente, a comunidade GitHub do Brasil tem crescido ano após ano em mais de 40%. O país abriga agora mais de 2 milhões de desenvolvedores, incluindo dezenas de milhares de contribuintes de open source.

Hora de comemorar e aprender

Para celebrar este crescimento e este momento, junte-se a nós no dia 2 de junho de 2022 às 18h BRT para um encontro virtual entre mantenedores locais e aprenda mais sobre a manutenção do open source, incluindo dicas para estabelecer seu perfil no Sponsors. No dia 3 de junho, também estaremos no TDC Innovations para falar sobre o GitHub Sponsors, nossas ferramentas de segurança, e muito mais. Junte-se a nós!

Vem mais por aí

Logo após a disponibilidade na Índia, o Sponsors no Brasil é mais um passo para melhorar o suporte ao ecossistema global de open source. Ainda há mais a ser feito, então fique atento enquanto continuamos a expandir e aprimorar a experiência do Sponsors.

The world runs on open source. That’s why three years ago we launched GitHub Sponsors so that open source developers and organizations could receive financial support for the important work they do every day.

As the open source community continues to grow—a quick to the 10 million new developers who have joined GitHub in the last 6 months!—we want to make sure that every developer has equal opportunities.

That’s why I’m excited to announce that GitHub Sponsors is now available in Brazil—an exciting expansion for one of our fastest growing developer communities. Developers already on the waitlist are now in the program and those with bank accounts in Brazil can create profiles to begin receiving funding.

Brazil’s open source community

While growth is happening globally, Brazil’s GitHub community has grown year-over-year by over 40%. The country is now home to more than 2 million developers, including tens of thousands of open source contributors.

Time to celebrate and learn

To celebrate this growth and momentum, join us on June 2at 6 pmBRT for a virtual meetup to meet local maintainers, learn more about maintaining open source, including tips for setting up your GitHub Sponsor profile. On June 3, we’ll also be at TDC Innovations conference to talk about GitHub Sponsors, our security tooling, and more. Join us!

More to come

Off the heels of availability in India, GitHub Sponsors in Brazil are another step towards improving support for the global open source ecosystem. There’s still more to be done, so stay tuned as we continue to expand and enhance the GitHub Sponsors experience.

推荐一个 Vibe Coding 工具： Claudia

guojam:

最近在用 Claude Code ，但命令行操作有点繁琐，偶然发现了 Claudia 这个开源项目，给 Claude Code 加了个 GUI 界面。 主要功能

数据全部本地存储，不上云 多项目管理，一键切换 Token 使用监控，避免超额 会话历史查看 可以给不同项目配置不同的 AI 助手

基于 Tauri + React + Rust ，跨平台支持。 使用感受 之前用 Claude Code 需要频繁敲命令，现在有了界面操作方便多了。项目切换、历史查看都很直观，Token 监控也实用。 关于下载 官方只提供源码，没有编译好的二进制版本。我自己编译了一份，放在 GitHub Release 里，需要的可以直接下载。 https://github.com/GoJam11/claudia-macos 官网 https://claudiacode.com/ 有在用 Claude Code 的 V 友吗？这个工具确实能提升不少体验。
